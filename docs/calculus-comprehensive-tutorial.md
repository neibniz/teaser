## 1. 微积分到底研究什么

微积分研究两个核心问题：

```text
变化率：一个量正在怎样变化？
累积量：许多微小变化合起来是多少？
```

导数回答变化率问题。

积分回答累积量问题。

例如：

```text
位置 s(t) 随时间变化
导数 v(t) = s'(t) 表示速度
积分 ∫ v(t) dt 表示位移累积
```

这不是两个孤立工具，而是一对互逆思想：

```text
导数：从总量看局部变化
积分：从局部变化重建总量
```

### 1.1 为什么微积分重要

现实世界里，很多量不是静止的：

- 速度在变化。
- 温度在变化。
- 股票价格在变化。
- 神经网络损失函数随参数变化。
- 流体速度在空间中变化。
- 概率密度在区间上累积为概率。
- 算法复杂度可以用连续近似理解增长趋势。

微积分提供一种语言：

```text
把变化拆得足够小；
在小范围内用简单模型近似；
再把无数小块累积回来。
```

这是微积分最深层的思想。

更具体地说，微积分的学习对象不是一个个孤立公式，而是三种关系：

```text
局部和整体的关系
离散和连续的关系
近似和精确的关系
```

例如，速度计显示的是“此刻速度”，但路程表显示的是“已经走过的总距离”。如果只知道每一刻速度，怎样恢复总距离？这就是积分。如果只知道路程随时间的变化记录，怎样知道某一刻速度？这就是导数。

再例如，一个曲线下面的面积看起来是连续的、弯曲的、难以直接计算的。但如果把它切成很多小矩形，每个矩形面积都很容易算。当矩形越来越窄时，近似会越来越好。微积分最重要的勇气就在这里：

```text
先用粗糙但可计算的小块近似；
再用极限把近似推向精确。
```

因此，微积分不是“把无穷小当成魔法”，而是用极限把直觉中的无穷小过程变成可验证的数学对象。

### 1.2 微积分的两种眼睛

学习时要同时训练两种视角：

| 视角 | 关注问题 |
| --- | --- |
| 几何视角 | 图像长什么样？斜率、面积、曲面、方向是什么？ |
| 代数视角 | 公式如何变形？极限、求导、积分如何计算？ |
| 物理视角 | 量纲是什么？速度、功、质量、流量如何解释？ |
| 严谨视角 | 这个结论需要什么条件？有没有反例？ |

优秀的学生不会只问：

```text
这道题套哪个公式？
```

他们会先问：

```text
这个问题要计算变化率、累积量、近似、极值，还是空间中的流动？
图像上发生了什么？
如果把条件改掉，结论还成立吗？
```

一个好的学习习惯是给每个新概念建立“多重翻译”：

| 概念 | 代数翻译 | 几何翻译 | 应用翻译 |
| --- | --- | --- | --- |
| 极限 | 函数值趋近某个数 | 图像靠近某高度 | 逼近过程是否稳定 |
| 导数 | 差商极限 | 切线斜率 | 瞬时速度、边际变化 |
| 积分 | Riemann 和极限 | 有符号面积 | 总量累积 |
| 级数 | 部分和极限 | 无限累加是否稳定 | 近似复杂函数 |
| 梯度 | 偏导数组成的向量 | 最快上升方向 | 优化更新方向 |

如果一个知识点只能用公式说出来，还没有真正理解。至少要能用一句图像语言和一句应用语言解释它。

## 2. 学习微积分的核心思维

### 2.1 四问法

每学一个知识点，问四个问题：

```text
1. 它要解决什么问题？
2. 如果没有它，我们会卡在哪里？
3. 它的直觉图像是什么？
4. 它的严格条件是什么？
```

例如学习导数时：

```text
问题：如何描述某一瞬间的速度？
困难：平均速度依赖时间间隔，不是瞬时。
直觉：割线斜率在两点靠近时趋向切线斜率。
严格：差商极限存在。
```

这比记住 `f'(x)` 的公式更重要。

### 2.2 小量思想

微积分常常把复杂对象切成小块。

例如求曲线下面积：

```text
把区间分成很多小段
每小段用矩形近似
矩形面积相加
让小段宽度趋近 0
得到精确面积
```

这种思想在很多地方重复出现：

| 问题 | 小量思想 |
| --- | --- |
| 导数 | 用很小的横向变化观察纵向变化 |
| 积分 | 用许多小面积累积成总面积 |
| 弧长 | 用许多小线段逼近曲线 |
| 体积 | 用许多薄片累积 |
| 曲面积分 | 用许多小平面片逼近曲面 |
| 微分方程 | 用局部变化规律推出整体演化 |

### 2.3 局部线性化

微积分的一个核心洞察是：

```text
足够局部地看，很多光滑曲线像一条直线。
```

例如：

```text
f(x + Δx) ≈ f(x) + f'(x) Δx
```

这就是线性近似。

它解释了为什么导数重要：

```text
导数给出函数在某点附近最好的线性近似斜率。
```

在多变量中，局部线性化变成：

```text
f(a + h) ≈ f(a) + ∇f(a) · h
```

这也是机器学习中梯度下降、牛顿法、误差传播的基础。

### 2.4 极限思想

微积分中很多对象不是直接定义出来的，而是通过逼近定义：

```text
瞬时速度 = 平均速度的极限
面积 = 矩形和的极限
曲线长度 = 折线长度的极限
无穷级数和 = 部分和的极限
```

所以极限不是微积分的一个章节，而是微积分的底层操作系统。

### 2.5 一种高质量学习模板

每遇到一个新公式，不要立刻做题，而是按下面顺序审问它：

```text
1. 这个公式从哪个问题来？
2. 公式里的每一项代表什么量？
3. 哪一步用了极限、连续、可导、可积等条件？
4. 如果某个条件不满足，会发生什么？
5. 能不能用一个简单例子验证？
6. 能不能画图解释？
```

以乘积法则为例：

```text
(fg)' = f'g + fg'
```

普通记忆方式是背公式。更好的方式是想象一个长方形，长为 `f`，宽为 `g`。长和宽都变一点点时，面积变化来自两条边的主要贡献：

```text
长变了：df * g
宽变了：f * dg
角落小块：df * dg
```

当取一阶变化率时，`df * dg` 是更高阶小量，被极限过程压掉。这时公式就不再是符号，而是一个可以重新推出来的思想。

## 3. 预备知识：函数、图像与变化

### 3.1 函数是什么

函数是输入到输出的规则：

```text
f: x -> f(x)
```

例如：

```text
f(x) = x^2
```

输入 `x`，输出 `x^2`。

微积分关心的不是只算一个点，而是研究：

```text
当 x 改变时，f(x) 如何改变？
```

### 3.2 定义域和值域

函数一定要关注定义域。

例如：

```text
f(x) = 1 / (x - 2)
```

在 `x = 2` 处没有定义。

如果忽略定义域，很多微积分判断都会出错。

常见问题：

```text
极限可以存在但函数值不存在吗？
函数在某点连续需要函数值存在吗？
求导前要先确认函数在该点附近有定义吗？
```

答案分别是：

```text
可以。
需要。
需要。
```

### 3.3 图像语言

函数图像是微积分直觉的来源。

要学会从图像上读出：

| 图像特征 | 微积分含义 |
| --- | --- |
| 上升 | 导数为正 |
| 下降 | 导数为负 |
| 越来越陡 | 导数绝对值变大 |
| 向上弯 | 二阶导数为正 |
| 向下弯 | 二阶导数为负 |
| 峰顶或谷底 | 可能有局部极值 |
| 断点 | 可能不连续 |
| 尖角 | 可能不可导 |
| 曲线下面积 | 积分 |

优秀学生解题时，经常先画一张粗略图。

不是为了画得漂亮，而是为了知道问题应该往哪个方向走。

### 3.4 函数组合

很多复杂函数是由简单函数组合而来：

```text
sin(x^2)
e^(3x+1)
ln(1+x^2)
```

学习微积分要有“拆结构”的习惯。

例如：

```text
f(x) = sin(x^2)
外层：sin(u)
内层：u = x^2
```

链式法则就是针对这种结构。

### 3.5 函数题的深度阅读法

初学者看到函数时常常只想“怎么算”。更成熟的做法是先读函数的结构。

例如：

```text
f(x) = ln(1 + x^2)
```

你应该立即读出：

```text
外层是 ln(u)，要求 u > 0。
内层是 1 + x^2，总是 >= 1。
所以定义域是所有实数。
函数是偶函数，因为 x^2 不变。
x=0 时取最小值 ln 1 = 0。
当 |x| 变大时，函数缓慢增长。
```

这类阅读能力很重要，因为很多微积分题并不是难在计算，而是难在不知道从哪里入手。读函数时至少检查五件事：

```text
定义域
对称性
零点或特殊点
增长趋势
组合结构
```

例如 `e^{-x^2}` 不是普通指数函数，而是“指数函数 + 负平方”。它总是正的，左右对称，在 `x=0` 最大，向两边快速衰减。这些信息会直接影响求导、积分、概率密度和图像判断。

## 4. 极限：无限接近时会发生什么

### 4.1 为什么需要极限

考虑函数：

```text
f(x) = (x^2 - 1) / (x - 1)
```

在 `x = 1` 处，表达式变成 `0/0`，没有定义。

但当 `x` 接近 1：

```text
(x^2 - 1) / (x - 1)
= (x - 1)(x + 1) / (x - 1)
= x + 1, x ≠ 1
```

所以当 `x` 趋近 1 时，函数值趋近 2。

写作：

```text
lim_{x -> 1} (x^2 - 1) / (x - 1) = 2
```

注意：

```text
极限关心 x 靠近某点时的趋势，不一定关心该点函数值本身。
```

### 4.2 左极限与右极限

```text
lim_{x -> a^-} f(x)
lim_{x -> a^+} f(x)
```

分别表示从左边和右边接近。

两侧极限都存在且相等时，双侧极限存在：

```text
lim_{x -> a} f(x) = L
```

如果左右极限不同，双侧极限不存在。

例子：

```text
f(x) = |x| / x
```

当 `x -> 0^-`：

```text
f(x) = -1
```

当 `x -> 0^+`：

```text
f(x) = 1
```

所以：

```text
lim_{x -> 0} |x| / x 不存在
```

### 4.3 极限不等于代入

如果函数在附近表现良好，极限常常可以代入。

但不是所有极限都能直接代入。

例如：

```text
lim_{x -> 0} sin x / x = 1
```

直接代入是 `0/0`，需要几何证明、夹逼定理或泰勒展开。

初学者要形成习惯：

```text
先代入看看类型；
如果得到正常数值，通常结束；
如果得到 0/0、∞/∞、0·∞、∞-∞ 等未定式，就需要变形。
```

### 4.4 常见未定式

| 形式 | 可能处理方法 |
| --- | --- |
| `0/0` | 因式分解、有理化、三角恒等式、洛必达、泰勒 |
| `∞/∞` | 除最高次、洛必达、渐近比较 |
| `0·∞` | 改写成分式 |
| `∞ - ∞` | 通分、有理化、提取主项 |
| `1^∞` | 取对数 |
| `0^0` | 取对数或转化 |
| `∞^0` | 取对数或转化 |

### 4.5 夹逼定理

如果：

```text
g(x) <= f(x) <= h(x)
```

并且：

```text
lim g(x) = lim h(x) = L
```

那么：

```text
lim f(x) = L
```

经典应用：

```text
lim_{x -> 0} x^2 sin(1/x) = 0
```

因为：

```text
-1 <= sin(1/x) <= 1
```

所以：

```text
-x^2 <= x^2 sin(1/x) <= x^2
```

两边都趋向 0，中间也趋向 0。

这道题的启发：

```text
高频振荡不一定破坏极限；
如果振幅被压到 0，整体仍可收敛。
```

### 4.6 无穷远处的极限

例如：

```text
lim_{x -> ∞} (3x^2 + 1) / (x^2 - 5)
```

除以最高次 `x^2`：

```text
(3 + 1/x^2) / (1 - 5/x^2) -> 3
```

直觉：

```text
x 很大时，低次项相对高次项可以忽略。
```

但要注意，不是所有函数都能只看最高次。

例如：

```text
e^x, x^100, ln x
```

增长速度不同：

```text
ln x << x^a << b^x << x! << x^x
```

### 4.7 ε-δ 定义的意义

严格定义：

```text
lim_{x -> a} f(x) = L
```

表示：

```text
对任意 ε > 0，都存在 δ > 0，使得当 0 < |x-a| < δ 时，有 |f(x)-L| < ε。
```

不要被符号吓住。

它表达的是一个挑战游戏：

```text
对方指定输出误差 ε；
你要找到输入距离 δ；
只要 x 离 a 足够近，f(x) 就离 L 足够近。
```

这个定义的价值是排除模糊说法。

“越来越接近”在直觉上容易理解，但数学需要可验证的标准。

### 4.8 极限题的深度拆解法

极限题的核心不是技巧堆叠，而是判断“主导行为”。

遇到：

```text
lim_{x -> a} f(x)
```

可以按下面顺序思考：

```text
1. a 是否在定义域内？
2. 直接代入得到什么？
3. 如果是正常数值，是否可以用连续性直接结束？
4. 如果是未定式，哪两部分在竞争？
5. 哪一部分是主导项？
6. 是否需要分左右、分路径或分段讨论？
```

例子：

```text
lim_{x -> 0} (1 - cos x) / x^2
```

直接代入是 `0/0`。这说明分子和分母都趋向 0，需要比较谁更快。用常见近似：

```text
cos x ≈ 1 - x^2/2
```

所以：

```text
1 - cos x ≈ x^2/2
```

于是极限应为：

```text
1/2
```

这种做法背后的思想是：

```text
在极限点附近，用最主要的低阶项替代复杂函数。
```

再看一个例子：

```text
lim_{x -> 0} sin(5x) / sin(2x)
```

因为小角度下 `sin u ≈ u`：

```text
sin(5x) ≈ 5x
sin(2x) ≈ 2x
```

所以极限是：

```text
5/2
```

优秀学生会在心里同时保留两层解释：

```text
计算层：用等价无穷小或标准极限。
直觉层：小角度下正弦函数几乎等于输入本身。
```

### 4.9 ε-δ 定义的一个完整例子

证明：

```text
lim_{x -> 2} (3x + 1) = 7
```

按照定义，要让：

```text
|(3x+1) - 7| < ε
```

化简：

```text
|3x - 6| = 3|x-2| < ε
```

所以只要：

```text
|x-2| < ε/3
```

就够了。

因此可以选：

```text
δ = ε/3
```

证明结构是：

```text
给定任意 ε > 0，取 δ = ε/3。
若 0 < |x-2| < δ，则 |(3x+1)-7| = 3|x-2| < 3δ = ε。
```

这类证明的思路是“反向设计”：

```text
先看目标 |f(x)-L| < ε；
把它变形成对 |x-a| 的要求；
再选择 δ。
```

它训练的是严谨性，不是为了日常计算每道极限都这样写。

## 5. 连续性：没有突然断裂的变化

函数在 `x = a` 连续，需要满足三件事：

```text
1. f(a) 有定义。
2. lim_{x -> a} f(x) 存在。
3. lim_{x -> a} f(x) = f(a)。
```

### 5.1 连续的直觉

连续函数的图像可以粗略理解为：

```text
在局部没有跳跃、洞或无穷断裂。
```

但这只是直觉，不是严格定义。

严格定义依靠极限。

### 5.2 三类常见不连续

| 类型 | 特征 | 例子 |
| --- | --- | --- |
| 可去不连续 | 极限存在，但函数值缺失或不等于极限 | `(x^2-1)/(x-1)` 在 `x=1` |
| 跳跃不连续 | 左右极限存在但不相等 | 阶跃函数 |
| 无穷不连续 | 函数值趋向无穷 | `1/x` 在 `x=0` |

### 5.3 连续函数的重要定理

#### 介值定理

如果 `f` 在 `[a,b]` 连续，且 `N` 在 `f(a)` 与 `f(b)` 之间，那么存在 `c`：

```text
f(c) = N
```

直觉：

```text
连续曲线从低处走到高处，不能跳过中间高度。
```

#### 极值定理

如果 `f` 在闭区间 `[a,b]` 连续，那么 `f` 一定取得最大值和最小值。

注意条件：

- 闭区间。
- 有界区间。
- 连续。

如果缺一个条件，结论可能失败。

例如：

```text
f(x) = x 在 (0,1) 上没有最大值和最小值。
```

### 5.4 连续性题的判定流程

判断函数在一点是否连续，不要凭图像直觉直接下结论，按三步走：

```text
1. 函数值 f(a) 是否存在？
2. 极限 lim_{x -> a} f(x) 是否存在？
3. 二者是否相等？
```

例子：

```text
f(x) = {
  (x^2 - 1)/(x - 1), x ≠ 1
  3,                 x = 1
}
```

在 `x=1`：

```text
f(1)=3
lim_{x -> 1} (x^2-1)/(x-1)=2
```

函数值存在，极限也存在，但不相等，所以不连续。

如果把 `f(1)` 改成 2，函数就连续。这说明可去不连续的本质是：

```text
附近趋势是好的，只是该点函数值没有填对。
```

连续性在后面很重要，因为许多定理依赖它：

```text
微积分基本定理要求被积函数有足够好的连续性。
介值定理依赖连续。
闭区间极值定理依赖连续。
很多换元和极限代入依赖连续。
```

所以看到题目使用某个定理前，要先问：

```text
函数在相关区间上连续吗？
区间是闭的吗？
区间有界吗？
```

## 6. 导数：瞬时变化率

### 6.1 从平均速度到瞬时速度

平均速度：

```text
(s(t+h) - s(t)) / h
```

当 `h` 趋近 0，就得到瞬时速度：

```text
s'(t) = lim_{h -> 0} (s(t+h) - s(t)) / h
```

对一般函数：

```text
f'(a) = lim_{h -> 0} (f(a+h) - f(a)) / h
```

这个差商有几何意义：

```text
割线斜率 -> 切线斜率
```

### 6.2 导数的三种理解

| 理解方式 | 含义 |
| --- | --- |
| 几何 | 曲线在某点的切线斜率 |
| 物理 | 瞬时变化率，如速度、边际成本 |
| 近似 | 函数在该点附近的最佳线性近似斜率 |

导数不是一个符号游戏。

看到 `f'(a)` 时，要能想到：

```text
x 增加一点点，f(x) 大约增加 f'(a) 倍的一点点。
```

### 6.3 用定义求导

例子：

```text
f(x) = x^2
```

用定义：

```text
f'(x) = lim_{h -> 0} ((x+h)^2 - x^2) / h
      = lim_{h -> 0} (2xh + h^2) / h
      = lim_{h -> 0} (2x + h)
      = 2x
```

不要只记 `x^2` 的导数是 `2x`。

要理解：

```text
(x+h)^2 - x^2 = 2xh + h^2
```

其中：

- `2xh` 是主要变化。
- `h^2` 是更小的二阶变化。
- 除以 `h` 后，`h` 趋于 0，二阶小量消失。

这就是线性近似的来源。

### 6.4 可导一定连续

如果函数在某点可导，那么它在该点连续。

直觉：

```text
如果有良好切线，函数不能在该点跳跃。
```

但反过来不成立。

例如：

```text
f(x) = |x|
```

在 `x = 0` 连续，但不可导。

左斜率为 `-1`，右斜率为 `1`。

图像有尖角。

### 6.5 不可导的常见原因

| 原因 | 例子 |
| --- | --- |
| 尖角 | `|x|` 在 0 |
| 垂直切线 | `x^(1/3)` 在 0 |
| 不连续 | 阶跃函数 |
| 高频振荡 | `x sin(1/x)` 在 0 附近需仔细判断 |

遇到不可导问题时，问：

```text
左导数和右导数是否相等？
差商是否有极限？
函数是否先连续？
```

### 6.6 导数的深层解释：一阶信息

导数是一阶信息。

所谓一阶信息，是指当 `h` 很小时：

```text
f(a+h) - f(a)
```

主要由哪一项决定。

如果函数可导，则：

```text
f(a+h) - f(a) = f'(a)h + 比 h 更小的误差
```

这里“比 h 更小”可以理解为误差除以 `h` 后趋向 0。

这句话比公式更重要，因为它解释了：

- 为什么导数能做线性近似。
- 为什么切线能代表局部曲线。
- 为什么优化时 `f'(x)=0` 是关键候选条件。
- 为什么牛顿法和梯度下降能用局部信息推动全局搜索。

例子：

```text
f(x)=x^2, a=3
```

小变化：

```text
f(3+h)-f(3) = (3+h)^2 - 9 = 6h + h^2
```

当 `h` 很小时：

```text
6h 是主要变化
h^2 是更小误差
```

所以：

```text
f'(3)=6
```

这说明导数不是“某个求导公式的输出”，而是“局部变化中最主要的线性部分”。

## 7. 求导规则：把复杂变化拆成基本变化

求导规则不是魔法，而是帮助我们避免每次都回到极限定义。

### 7.1 常数、幂函数、指数和对数

常见导数：

```text
d/dx c = 0
d/dx x^n = n x^(n-1)
d/dx e^x = e^x
d/dx ln x = 1/x
d/dx sin x = cos x
d/dx cos x = -sin x
```

每个公式都应该能说出直觉：

- 常数不变化，所以导数为 0。
- `x^n` 的主要变化来自 `n` 个位置各贡献一个 `h`。
- `e^x` 是变化率等于自身的函数。
- `ln x` 增长越来越慢，导数 `1/x` 越来越小。

### 7.2 加法和常数倍

```text
(f + g)' = f' + g'
(cf)' = c f'
```

直觉：

```text
两个变化量相加，总变化率也相加。
```

### 7.3 乘积法则

```text
(fg)' = f'g + fg'
```

为什么？

小变化：

```text
(f + df)(g + dg) - fg
= f dg + g df + df dg
```

其中 `df dg` 是二阶小量，在一阶导数里消失。

所以：

```text
d(fg) ≈ f dg + g df
```

这比死记公式更可靠。

### 7.4 商法则

```text
(f/g)' = (f'g - fg') / g^2
```

可以从乘积法则和链式法则推出：

```text
f/g = f * g^(-1)
```

不要把商法则当成孤立公式。

### 7.5 链式法则

```text
d/dx f(g(x)) = f'(g(x)) g'(x)
```

直觉：

```text
x 变一点点；
g 变 g'(x) 倍的一点点；
f 对 g 的变化再放大 f'(g(x)) 倍。
```

例子：

```text
d/dx sin(x^2) = cos(x^2) * 2x
```

结构分析：

```text
外层：sin u -> cos u
内层：u = x^2 -> 2x
```

### 7.6 对数求导

当函数是复杂乘积、商、幂时，可以取对数。

例子：

```text
y = x^x, x > 0
```

取对数：

```text
ln y = x ln x
```

两边求导：

```text
y'/y = ln x + 1
```

所以：

```text
y' = x^x (ln x + 1)
```

对数求导的本质：

```text
把乘法变加法，把指数搬下来。
```

### 7.7 求导规则的选择流程

复杂求导题要先识别表达式结构，而不是从左到右硬算。

可以按下面顺序问：

```text
1. 最外层是什么运算？
2. 是和差、乘积、商，还是复合？
3. 有没有变量同时出现在底数和指数？
4. 有没有隐含的内层函数？
5. 能不能先化简再求导？
```

例子：

```text
f(x) = e^{x^2} ln(1+x)
```

最外层是乘积：

```text
左边：e^{x^2}
右边：ln(1+x)
```

左边又是复合函数：

```text
d/dx e^{x^2} = e^{x^2} * 2x
```

右边也是复合函数：

```text
d/dx ln(1+x) = 1/(1+x)
```

所以：

```text
f'(x) = 2x e^{x^2} ln(1+x) + e^{x^2}/(1+x)
```

这道题同时用到乘积法则和链式法则。很多初学者出错，是因为只看见局部函数，没有先判断最外层结构。

一个实用口诀：

```text
先看外层，再看内层；
先定结构，再做计算；
每一步都问：这一项来自哪个变化？
```

## 8. 隐函数、反函数与参数方程求导

### 8.1 隐函数求导

有些关系不是显式写成 `y = f(x)`。

例如圆：

```text
x^2 + y^2 = 1
```

两边对 `x` 求导：

```text
2x + 2y y' = 0
```

所以：

```text
y' = -x/y
```

注意：

```text
y 是 x 的函数，所以 d(y^2)/dx = 2y y'。
```

这就是链式法则。

### 8.2 反函数求导

如果 `y = f(x)`，反函数为 `x = f^{-1}(y)`。

反函数导数：

```text
(f^{-1})'(y) = 1 / f'(x)
```

直觉：

```text
原函数斜率很大，反过来横纵轴交换后斜率就很小。
```

例子：

```text
y = e^x
x = ln y
```

因为：

```text
dy/dx = e^x = y
```

所以：

```text
dx/dy = 1/y
```

即：

```text
d/dy ln y = 1/y
```

### 8.3 参数方程求导

如果：

```text
x = x(t)
y = y(t)
```

那么：

```text
dy/dx = (dy/dt) / (dx/dt)
```

前提是 `dx/dt ≠ 0`。

直觉：

```text
t 是背后的时间参数；
y 对 x 的变化 = y 对 t 的变化 / x 对 t 的变化。
```

### 8.4 三类特殊求导的共同本质

隐函数、反函数、参数方程看起来是三种技巧，其实都在处理同一个问题：

```text
变量之间的关系不是简单的 y = f(x)。
```

隐函数求导中，关系被写成：

```text
F(x,y)=0
```

反函数求导中，关系是：

```text
y=f(x)，但我们想反过来看 x 与 y 的关系。
```

参数方程中，关系由第三个变量控制：

```text
x=x(t), y=y(t)
```

它们共同依赖链式法则。

解题时可以这样判断：

| 题目形态 | 关键动作 |
| --- | --- |
| `x` 和 `y` 混在同一个方程中 | 把 `y` 当成 `x` 的函数，逐项求导 |
| 要求反函数导数 | 先找原函数斜率，再取倒数 |
| `x,y` 都由 `t` 给出 | 先分别求 `dx/dt`、`dy/dt`，再相除 |

检查点：

```text
隐函数求导后，答案通常含 x 和 y。
反函数求导要注意原函数导数不能为 0。
参数方程求 dy/dx 要注意 dx/dt 不能为 0。
```

## 9. 导数的核心应用

### 9.1 单调性

如果在区间上：

```text
f'(x) > 0
```

函数递增。

如果：

```text
f'(x) < 0
```

函数递减。

但要注意：

```text
f'(x) = 0 的点不一定是极值点。
```

例如：

```text
f(x) = x^3
```

在 `x = 0`，导数为 0，但函数仍然递增。

### 9.2 极值

局部极值可能出现在：

```text
f'(x) = 0
```

或导数不存在的点。

这些叫 critical points。

解题步骤：

```text
1. 找定义域。
2. 求 f'(x)。
3. 找 f'(x)=0 或 f' 不存在的点。
4. 用一阶导数变号、二阶导数或图像判断。
5. 如果是闭区间全局极值，还要检查端点。
```

### 9.3 一阶导数判别法

如果在 `c` 附近：

```text
f' 从 + 变 -
```

则 `c` 是局部最大值。

如果：

```text
f' 从 - 变 +
```

则 `c` 是局部最小值。

### 9.4 二阶导数与凹凸性

二阶导数：

```text
f''(x)
```

表示导数的变化率。

如果：

```text
f''(x) > 0
```

图像向上凹，像碗。

如果：

```text
f''(x) < 0
```

图像向下凹。

二阶导数判别法：

```text
f'(c)=0 且 f''(c)>0 -> 局部最小
f'(c)=0 且 f''(c)<0 -> 局部最大
f''(c)=0 -> 不能判断
```

### 9.5 优化问题

优化题不是先求导，而是先建模。

步骤：

```text
1. 明确要最大化或最小化的量。
2. 选变量。
3. 写目标函数。
4. 写约束关系。
5. 用约束减少变量。
6. 求导找候选点。
7. 检查边界和实际意义。
```

例子：

```text
用长度为 100 的篱笆围一个矩形，最大面积是多少？
```

设长 `x`，宽 `y`：

```text
2x + 2y = 100
y = 50 - x
```

面积：

```text
A(x) = x(50-x) = 50x - x^2
```

求导：

```text
A'(x) = 50 - 2x
```

令 `A'(x)=0`：

```text
x = 25
y = 25
```

最大面积为：

```text
625
```

直觉：

```text
固定周长的矩形中，正方形面积最大。
```

### 9.6 用导数画完整函数画像

导数应用不只是找最大最小值。一个函数的完整画像通常包括：

```text
定义域
截距
对称性
一阶导数符号
临界点
二阶导数符号
凹凸区间
拐点
端点或无穷远行为
```

例子：

```text
f(x)=x^3-3x
```

一阶导数：

```text
f'(x)=3x^2-3=3(x-1)(x+1)
```

临界点：

```text
x=-1, x=1
```

符号分析：

```text
x < -1: f'(x)>0，递增
-1 < x < 1: f'(x)<0，递减
x > 1: f'(x)>0，递增
```

所以：

```text
x=-1 是局部最大
x=1 是局部最小
```

二阶导数：

```text
f''(x)=6x
```

凹凸性：

```text
x<0 向下凹
x>0 向上凹
x=0 是拐点
```

这种分析比单纯求一个导数更重要，因为它把公式转换成函数行为。

## 10. 微分与线性近似

### 10.1 线性近似公式

在 `x = a` 附近：

```text
f(x) ≈ f(a) + f'(a)(x-a)
```

如果令 `dx = x-a`：

```text
df = f'(a) dx
```

### 10.2 为什么有用

例子：估算 `sqrt(4.1)`。

取：

```text
f(x)=sqrt(x), a=4
```

```text
f(4)=2
f'(x)=1/(2sqrt(x))
f'(4)=1/4
```

所以：

```text
sqrt(4.1) ≈ 2 + (1/4)(0.1) = 2.025
```

这说明导数给出局部变化的第一近似。

### 10.3 误差估计

线性近似忽略了二阶及更高阶变化。

如果函数弯曲很明显，误差会变大。

Taylor 公式会进一步说明：

```text
f(a+h) = f(a) + f'(a)h + f''(a)h^2/2 + ...
```

线性近似就是只保留前两项。

### 10.4 线性近似题的标准拆法

线性近似题可以机械地按四步走：

```text
1. 选择函数 f。
2. 选择靠近目标点、容易计算的 a。
3. 计算 f(a) 和 f'(a)。
4. 使用 f(a+h) ≈ f(a) + f'(a)h。
```

例子：估算 `1/1.98`。

选择：

```text
f(x)=1/x
a=2
h=-0.02
```

计算：

```text
f(2)=1/2
f'(x)=-1/x^2
f'(2)=-1/4
```

所以：

```text
1/1.98 ≈ 1/2 + (-1/4)(-0.02)
       = 0.5 + 0.005
       = 0.505
```

真实值约为 `0.50505...`。

这说明线性近似不是猜测，而是用切线替代曲线。

判断近似是否可靠时，问：

```text
目标点离 a 近不近？
函数在这段范围弯曲强不强？
是否需要二阶修正？
```

## 11. 积分：从累积量开始理解

### 11.1 面积问题

想求曲线 `y=f(x)` 在 `[a,b]` 下方的面积。

如果 `f` 是常数：

```text
面积 = 高 * 宽
```

如果 `f` 变化：

```text
把区间切成很多小段；
每段用一个矩形近似；
把矩形面积相加；
让最大段长趋近 0。
```

这就是定积分的来源。

### 11.2 Riemann 和

把 `[a,b]` 分成 `n` 段：

```text
Δx = (b-a)/n
```

取每段一点 `x_i*`：

```text
Σ f(x_i*) Δx
```

当 `n -> ∞`：

```text
∫_a^b f(x) dx
```

### 11.3 `dx` 的直觉

在初学阶段，可以把：

```text
f(x) dx
```

理解为：

```text
宽度为 dx 的小矩形面积。
```

严格上，`dx` 不是普通数，但这种小量直觉非常有用。

### 11.4 定积分不只是面积

定积分更一般地表示累积。

| 被积函数 | 积分含义 |
| --- | --- |
| 速度 | 位移 |
| 加速度 | 速度变化 |
| 密度 | 质量 |
| 力沿路径方向分量 | 功 |
| 概率密度 | 概率 |
| 流量率 | 总流量 |

核心：

```text
如果某个量的局部密度或变化率已知，总量通常由积分给出。
```

### 11.5 积分建模的深度理解

做积分应用题时，不要先想“套哪个积分公式”，而要问：

```text
我要累积什么小量？
这个小量的密度是什么？
这个小量的宽度、面积元或体积元是什么？
累积范围是什么？
```

例如，一个物体沿直线运动，速度为：

```text
v(t)=3t^2
```

从 `t=1` 到 `t=2` 的位移不是速度乘一个固定时间，因为速度在变。把时间切成小段：

```text
小位移 ≈ v(t) Δt
```

累积得到：

```text
位移 = ∫_1^2 3t^2 dt = [t^3]_1^2 = 7
```

同一个结构可以迁移到很多问题：

```text
小质量 = 密度 * 小长度/小面积/小体积
小功 = 力 * 小位移
小概率 = 概率密度 * 小区间长度
小体积 = 截面积 * 小厚度
```

这就是积分建模的核心：

```text
先写小量，再累积。
```

## 12. 微积分基本定理

微积分基本定理连接导数和积分。

### 12.1 第一部分：累积函数的导数

定义：

```text
F(x) = ∫_a^x f(t) dt
```

如果 `f` 连续，则：

```text
F'(x) = f(x)
```

直觉：

```text
当上限 x 增加一点 dx，新增面积大约是 f(x) dx。
所以累积面积的变化率就是当前高度 f(x)。
```

### 12.2 第二部分：用原函数算定积分

如果：

```text
F'(x) = f(x)
```

那么：

```text
∫_a^b f(x) dx = F(b) - F(a)
```

这非常强大。

本来定积分是无限多小矩形的极限，现在只要找原函数。

举一个完整例子：

```text
∫_0^3 2x dx
```

从面积角度看，这是曲线 `y=2x` 下的三角形面积：

```text
底 = 3
高 = 6
面积 = 1/2 * 3 * 6 = 9
```

从基本定理看，`2x` 的原函数是：

```text
F(x)=x^2
```

所以：

```text
∫_0^3 2x dx = F(3)-F(0)=9-0=9
```

两个答案一致。这个例子说明：

```text
积分的几何定义和原函数计算方法被基本定理连接起来。
```

### 12.3 为什么它深刻

导数看起来是局部操作：

```text
看某一点附近的变化率。
```

积分看起来是整体操作：

```text
累积整个区间的量。
```

基本定理说明：

```text
局部变化率和整体累积互相决定。
```

这是微积分的中心桥梁。

### 12.4 基本定理的常见误区

误区一：

```text
所有积分都能轻松找原函数。
```

实际上，很多函数没有初等原函数，例如：

```text
e^{-x^2}
sin(x)/x
```

它们仍然有定积分意义，只是不能用初等函数简单表示。

误区二：

```text
∫_a^b f(x) dx 一定是正面积。
```

定积分是有符号累积。函数在 x 轴下方时贡献为负。

误区三：

```text
F'(x)=f(x) 中的 F 可以随便选。
```

原函数可以相差常数，但定积分中常数会抵消：

```text
(F(b)+C) - (F(a)+C) = F(b)-F(a)
```

这解释了为什么不定积分需要 `+C`，而定积分计算时常数不影响结果。

## 13. 积分计算技巧

积分比求导难。

原因：

```text
求导有确定规则；
积分经常需要识别结构。
```

### 13.1 换元积分

换元是链式法则的逆过程。

例子：

```text
∫ 2x cos(x^2) dx
```

令：

```text
u = x^2
du = 2x dx
```

所以：

```text
∫ 2x cos(x^2) dx = ∫ cos u du = sin u + C = sin(x^2) + C
```

识别方法：

```text
看到复合函数 f(g(x))；
旁边又有 g'(x)；
考虑换元。
```

### 13.2 分部积分

分部积分来自乘积法则：

```text
(uv)' = u'v + uv'
```

积分后：

```text
∫ u dv = uv - ∫ v du
```

例子：

```text
∫ x e^x dx
```

取：

```text
u = x, dv = e^x dx
du = dx, v = e^x
```

得到：

```text
∫ x e^x dx = x e^x - ∫ e^x dx = x e^x - e^x + C
```

选择原则：

```text
让 u 求导后变简单；
让 dv 容易积分。
```

### 13.3 部分分式

有理函数积分常用部分分式。

例子：

```text
∫ 1/(x^2-1) dx
```

分解：

```text
1/(x^2-1) = 1/((x-1)(x+1))
          = 1/2 * 1/(x-1) - 1/2 * 1/(x+1)
```

所以：

```text
∫ 1/(x^2-1) dx
= 1/2 ln|x-1| - 1/2 ln|x+1| + C
```

### 13.4 三角代换

当出现：

```text
sqrt(a^2 - x^2)
sqrt(a^2 + x^2)
sqrt(x^2 - a^2)
```

可以考虑三角代换。

例如：

```text
x = a sin θ
```

因为：

```text
1 - sin^2 θ = cos^2 θ
```

三角代换的本质：

```text
利用恒等式把根号结构变简单。
```

### 13.5 积分技巧选择表

| 结构 | 优先考虑 |
| --- | --- |
| `f(g(x))g'(x)` | 换元 |
| 多项式乘指数/三角/对数 | 分部积分 |
| 有理函数 | 部分分式 |
| 根号二次式 | 三角代换或双曲代换 |
| 对称区间 | 奇偶性 |
| 周期函数 | 周期性质 |
| 很难求原函数 | 数值积分或级数 |

### 13.6 积分技巧的诊断流程

积分题最难的是选择方法。可以先做结构诊断：

```text
1. 能不能直接识别原函数？
2. 是否是复合函数及其导数？如果是，换元。
3. 是否是乘积，且一部分求导会变简单？如果是，分部积分。
4. 是否是有理函数？如果是，先因式分解再部分分式。
5. 是否有根号二次式？考虑三角代换。
6. 是否有对称区间？检查奇偶性。
7. 是否定积分很难但只需数值？考虑近似或计算工具。
```

例子：

```text
∫ x ln x dx
```

它是乘积。`ln x` 求导会变成 `1/x`，更简单；`x` 容易积分。所以用分部积分：

```text
u = ln x
dv = x dx
du = 1/x dx
v = x^2/2
```

得到：

```text
∫ x ln x dx = x^2 ln x / 2 - ∫ x/2 dx
            = x^2 ln x / 2 - x^2/4 + C
```

检查方法：

```text
对答案求导，看是否回到 x ln x。
```

积分题强烈建议养成“求导验算”的习惯。

## 14. 积分的几何、物理与概率应用

### 14.1 面积

两曲线之间面积：

```text
∫_a^b (上方函数 - 下方函数) dx
```

关键不是公式，而是：

```text
每一小条的高度是多少？
宽度是什么？
积分区间是什么？
```

### 14.2 体积：圆盘法和壳层法

绕 x 轴旋转：

```text
V = ∫ π [R(x)]^2 dx
```

如果有内外半径：

```text
V = ∫ π (R(x)^2 - r(x)^2) dx
```

壳层法：

```text
V = ∫ 2π * 半径 * 高度 dx
```

选择原则：

```text
切片后形状最简单的方法优先。
```

### 14.3 弧长

曲线 `y=f(x)` 在 `[a,b]` 上弧长：

```text
L = ∫_a^b sqrt(1 + (f'(x))^2) dx
```

直觉来自小直角三角形：

```text
ds ≈ sqrt(dx^2 + dy^2)
   = sqrt(1 + (dy/dx)^2) dx
```

### 14.4 功

如果力随位置变化：

```text
W = ∫ F(x) dx
```

例如弹簧力：

```text
F(x) = kx
```

从 0 拉到 a 的功：

```text
W = ∫_0^a kx dx = 1/2 ka^2
```

### 14.5 概率

连续随机变量密度函数 `p(x)`：

```text
P(a <= X <= b) = ∫_a^b p(x) dx
```

密度不是概率本身。

概率是密度在区间上的累积。

总概率：

```text
∫_{-∞}^{∞} p(x) dx = 1
```

### 14.6 应用题的统一建模模板

无论是面积、体积、功还是概率，都可以使用同一个模板：

```text
总量 = ∫ 局部密度 * 小单位
```

具体翻译：

| 问题 | 局部密度 | 小单位 | 总量 |
| --- | --- | --- | --- |
| 面积 | 高度 | `dx` | 面积 |
| 旋转体体积 | 截面积 | `dx` | 体积 |
| 质量 | 密度 | `dx`、`dA`、`dV` | 质量 |
| 功 | 力 | `dx` | 功 |
| 概率 | 概率密度 | `dx` | 概率 |

例子：一根长度为 10 的细杆，线密度为：

```text
ρ(x)=2+x
```

总质量是：

```text
M = ∫_0^10 (2+x) dx
  = [2x + x^2/2]_0^10
  = 20 + 50
  = 70
```

这个题不需要新公式。只要识别：

```text
小质量 = 线密度 * 小长度 = ρ(x) dx
```

很多积分应用题本质都是这一步。

## 15. 广义积分：无限区间和无界函数

### 15.1 无限区间

```text
∫_1^∞ 1/x^p dx
```

结果：

```text
p > 1 时收敛
p <= 1 时发散
```

推导：

```text
∫_1^b x^(-p) dx
```

再令 `b -> ∞`。

### 15.2 无界函数

例如：

```text
∫_0^1 1/sqrt(x) dx
```

虽然 `1/sqrt(x)` 在 0 附近无界，但积分收敛：

```text
∫_0^1 x^(-1/2) dx = 2
```

启发：

```text
函数值趋向无穷，不代表面积一定无穷。
```

### 15.3 比较判别

如果 `0 <= f(x) <= g(x)`：

- `g` 收敛，则 `f` 收敛。
- `f` 发散，则 `g` 发散。

核心思路：

```text
用已知函数控制未知函数。
```

这和夹逼定理是同一种思想。

### 15.4 广义积分的判断重点

广义积分的关键不是“上下限有无穷”这么表面，而是问：

```text
无限长的尾巴是否累积出有限量？
无界尖峰是否足够窄，从而面积有限？
```

两个模型必须熟练：

```text
∫_1^∞ 1/x^p dx: p > 1 收敛
∫_0^1 1/x^p dx: p < 1 收敛
```

注意两者方向相反。

原因是：

- 在无穷远处，衰减要足够快，`p > 1` 才收敛。
- 在 0 附近，爆炸不能太强，`p < 1` 才收敛。

例子：

```text
∫_0^1 1/x^2 dx
```

虽然区间有限，但在 0 附近爆炸太强，发散。

而：

```text
∫_1^∞ 1/x^2 dx
```

虽然区间无限，但尾部衰减足够快，收敛。

这提醒我们：

```text
区间无限不必然发散；
函数无界不必然发散；
要看累积效应。
```

## 16. 数列与级数：无限相加是否有意义

### 16.1 数列极限

数列：

```text
a_1, a_2, a_3, ...
```

如果 `n` 越来越大时 `a_n` 趋近 `L`，则：

```text
lim_{n -> ∞} a_n = L
```

### 16.2 级数

级数是无限求和：

```text
Σ a_n
```

严格上，它由部分和定义：

```text
s_N = a_1 + a_2 + ... + a_N
```

如果：

```text
lim_{N -> ∞} s_N
```

存在，则级数收敛。

### 16.3 几何级数

```text
Σ_{n=0}∞ ar^n
```

当 `|r| < 1`：

```text
sum = a / (1-r)
```

当 `|r| >= 1`，发散。

这是级数中最基础的模型。

### 16.4 必要条件

如果：

```text
Σ a_n 收敛
```

那么：

```text
a_n -> 0
```

但反过来不成立。

例子：

```text
Σ 1/n
```

项趋向 0，但级数发散。

### 16.5 常见判别法

| 判别法 | 适用场景 |
| --- | --- |
| 比较判别 | 与已知正项级数比较 |
| 极限比较 | 主项相同或相近 |
| 比值判别 | 阶乘、指数、幂混合 |
| 根值判别 | `n` 次幂结构 |
| 交错级数判别 | 符号交替且绝对值递减趋 0 |
| 积分判别 | 正、递减、连续函数对应 |

### 16.6 绝对收敛与条件收敛

如果：

```text
Σ |a_n| 收敛
```

则 `Σ a_n` 绝对收敛。

如果 `Σ a_n` 收敛但 `Σ |a_n|` 发散，叫条件收敛。

例如：

```text
Σ (-1)^(n+1) / n
```

条件收敛。

这种区别重要，因为条件收敛级数对重排很敏感。

### 16.7 级数判别法的选择思路

级数题最常见的问题是“不知道用哪个判别法”。可以按项的形状判断：

```text
1. 项是否不趋于 0？如果不趋于 0，直接发散。
2. 是否像 1/n^p？用 p 级数。
3. 是否像几何级数 ar^n？用几何级数。
4. 是否有阶乘或指数？优先比值判别。
5. 是否有 n 次幂整体？优先根值判别。
6. 是否正负交替？检查交错级数判别和绝对收敛。
7. 是否能和已知级数比较？用比较或极限比较。
```

例子：

```text
Σ n / 2^n
```

有指数 `2^n`，也有多项式 `n`。指数通常支配多项式。用比值判别：

```text
a_n = n/2^n
a_{n+1}/a_n = ((n+1)/2^{n+1}) / (n/2^n)
            = (n+1)/(2n)
            -> 1/2
```

因为极限小于 1，所以收敛。

直觉是：

```text
分母指数增长比 numerator 的线性增长快得多。
```

再看：

```text
Σ 1/(n ln n), n >= 2
```

它不像普通 p 级数。可以用积分判别，因为对应函数正、递减，并且：

```text
∫ 1/(x ln x) dx = ln(ln x)
```

趋向无穷，所以级数发散。

这类例子训练你看“增长速度”和“主导行为”。

## 17. 幂级数与泰勒展开

### 17.1 幂级数

幂级数形如：

```text
Σ c_n (x-a)^n
```

它像“无限次多项式”。

多项式容易求导、积分、近似。

在收敛并且确实收敛到原函数的区间内，幂级数可以把复杂函数表示成多项式极限。

### 17.2 收敛半径

幂级数通常在某个区间内收敛：

```text
|x-a| < R
```

`R` 叫收敛半径。

端点需要单独检查。

### 17.3 Taylor 公式

如果函数在 `a` 附近有各阶导数，可以写出它在 `a` 点的 Taylor 级数形式：

```text
f(a) + f'(a)(x-a) + f''(a)(x-a)^2/2! + ...
```

一般项：

```text
Σ f^(n)(a) (x-a)^n / n!
```

要特别注意：

```text
有 Taylor 级数，不自动等于函数本身。
```

真正能写成：

```text
f(x) = Σ f^(n)(a) (x-a)^n / n!
```

还需要 Taylor 余项在相应区间内趋近 0。换句话说，函数要在该区间内被自己的 Taylor 级数表示。

反例意识很重要：

```text
无限可导不等于一定能由 Taylor 级数恢复。
```

### 17.4 常见展开

```text
e^x = 1 + x + x^2/2! + x^3/3! + ...
sin x = x - x^3/3! + x^5/5! - ...
cos x = 1 - x^2/2! + x^4/4! - ...
ln(1+x) = x - x^2/2 + x^3/3 - ..., -1 < x <= 1
1/(1-x) = 1 + x + x^2 + x^3 + ..., |x|<1
```

前三个展开对所有实数 `x` 都成立；后两个有收敛区间限制。

### 17.5 Taylor 的直觉

Taylor 展开不是背公式。

它的思想是：

```text
在点 a 附近，用一个多项式让函数值、斜率、二阶变化、三阶变化... 尽量匹配。
```

一阶 Taylor 是线性近似。

二阶 Taylor 加入曲率。

更高阶 Taylor 加入更细节的局部形状。

### 17.6 误差项

实际使用 Taylor 时，要关心误差。

例如：

```text
e^x = 1 + x + x^2/2 + R_2(x)
```

如果只截断到二阶，需要知道 `R_2(x)` 多大。

优秀学生会问：

```text
我在哪个区间近似？
需要几位精度？
余项是否可控？
```

### 17.7 Taylor 例题拆解：为什么能做近似

例子：用 Taylor 估算 `sin(0.1)`。

在 `a=0` 处：

```text
sin x = x - x^3/3! + x^5/5! - ...
```

当 `x=0.1`：

```text
sin(0.1) ≈ 0.1 - 0.1^3/6
         = 0.1 - 0.000166666...
         = 0.099833333...
```

为什么只取两项就够好？

因为后面的项迅速变小：

```text
0.1^5/5! = 0.00001/120 ≈ 0.0000000833
```

这说明 Taylor 近似的质量取决于：

```text
x 离展开点 a 有多近；
函数高阶导数是否温和；
截断后的余项是否足够小。
```

再看 `e^x`：

```text
e^{0.1} ≈ 1 + 0.1 + 0.1^2/2 = 1.105
```

真实值约为 `1.10517`。

Taylor 方法的本质不是神奇记忆，而是：

```text
用多项式模仿函数在某点附近的局部行为。
```

### 17.8 Taylor 与线性近似、二阶近似的关系

线性近似：

```text
f(a+h) ≈ f(a) + f'(a)h
```

二阶近似：

```text
f(a+h) ≈ f(a) + f'(a)h + f''(a)h^2/2
```

Taylor 展开只是把这个过程继续下去。

学习时要把它们看成同一思想的不同精度版本：

| 近似 | 使用信息 | 能捕捉什么 |
| --- | --- | --- |
| 0 阶 | 函数值 | 附近大概高度 |
| 1 阶 | 函数值 + 斜率 | 局部直线趋势 |
| 2 阶 | 函数值 + 斜率 + 曲率 | 弯曲方向和弯曲程度 |
| 高阶 | 更多高阶导数 | 更细局部形状 |

如果题目只要求粗略估算，线性近似可能足够；如果要求误差更小，就考虑二阶或更高阶。

## 18. 微分方程入门

微分方程是包含未知函数及其导数的方程。

例如：

```text
dy/dt = ky
```

它描述：

```text
变化率与当前量成正比。
```

### 18.1 指数增长

方程：

```text
dy/dt = ky
```

解：

```text
y(t) = Ce^(kt)
```

如果 `k > 0`，增长。

如果 `k < 0`，衰减。

### 18.2 分离变量

例子：

```text
dy/dx = xy
```

分离：

```text
dy/y = x dx
```

这一步默认 `y ≠ 0`。原方程还有零解 `y = 0`，最后写成 `y = Ce^(x^2/2)` 时可以用 `C = 0` 把它包含进去。

积分：

```text
ln|y| = x^2/2 + C
```

所以：

```text
y = Ce^(x^2/2)
```

### 18.3 初值问题

微分方程通常需要初始条件。

例如：

```text
dy/dt = ky, y(0)=y0
```

得到：

```text
y(t)=y0 e^(kt)
```

初值决定具体解。

### 18.4 建模意识

看到“变化率与某个量有关”，就可以考虑微分方程。

例如：

- 放射性衰变。
- 人口增长。
- 冷却过程。
- 贷款和利息。
- 神经网络参数更新。

### 18.5 微分方程的深度理解：从规则到轨迹

普通方程求的是一个数。

例如：

```text
x^2 = 4
```

解是：

```text
x = 2 或 x = -2
```

微分方程求的是一个函数。

例如：

```text
dy/dt = ky
```

它没有直接告诉你 `y(t)` 是多少，而是告诉你：

```text
每一刻的变化率由当前状态决定。
```

这是一种“局部演化规则”。解微分方程就是从局部规则恢复整体轨迹。

以冷却问题为例：

```text
dT/dt = -k(T - T_env)
```

含义是：

```text
温度变化率与当前温度和环境温度的差成正比；
差越大，冷却越快；
差越小，冷却越慢。
```

这比单纯解出公式更重要，因为它训练建模能力。

微分方程题通常按下面流程：

```text
1. 明确未知函数是什么。
2. 找到变化率。
3. 用题目语言写出变化率和状态的关系。
4. 加入初始条件。
5. 解方程。
6. 检查解是否符合直觉和量纲。
```

## 19. 多变量函数：从曲线到曲面

单变量函数：

```text
y = f(x)
```

多变量函数：

```text
z = f(x,y)
```

图像从平面曲线变成空间曲面。

### 19.1 等高线

对 `z=f(x,y)`，等高线是：

```text
f(x,y)=c
```

它表示函数值相同的点。

地图上的海拔线就是等高线。

等高线帮助我们在二维平面上理解三维曲面。

### 19.2 极限与连续

多变量极限更复杂。

单变量只能从左、右接近。

多变量可以沿无数路径接近。

如果不同路径得到不同极限，则极限不存在。

例子：

```text
f(x,y) = xy / (x^2 + y^2)
```

沿 `y=x`：

```text
f(x,x)=x^2/(2x^2)=1/2
```

沿 `y=0`：

```text
f(x,0)=0
```

所以在 `(0,0)` 极限不存在。

启发：

```text
多变量极限不能只检查一条路径。
```

### 19.3 多变量函数的思维升级

从单变量到多变量，最大的变化不是公式变多，而是“方向”变多。

单变量中，靠近 `a` 只有两个方向：

```text
左边
右边
```

多变量中，靠近 `(a,b)` 有无数路径：

```text
直线
曲线
抛物线
螺旋线
任意路径
```

所以多变量极限更难。证明极限不存在，只要找到两条路径给出不同结果；但证明极限存在，不能只检查几条路径，通常要用估计、夹逼或极坐标。

例如：

```text
f(x,y)= (x^2 y^2)/(x^2+y^2)
```

因为：

```text
0 <= x^2 y^2/(x^2+y^2)
```

又有：

```text
x^2 <= x^2+y^2
```

所以：

```text
x^2 y^2/(x^2+y^2) <= y^2
```

当 `(x,y)->(0,0)` 时，`y^2 -> 0`，因此极限为 0。

这类题的核心是：

```text
把复杂表达式压在一个明显趋于 0 的函数下面。
```

## 20. 偏导数、梯度与方向导数

### 20.1 偏导数

偏导数固定其他变量，只看一个方向变化：

```text
∂f/∂x
∂f/∂y
```

例子：

```text
f(x,y)=x^2 y + sin y
```

对 `x`：

```text
∂f/∂x = 2xy
```

对 `y`：

```text
∂f/∂y = x^2 + cos y
```

### 20.2 梯度

梯度：

```text
∇f = (∂f/∂x, ∂f/∂y)
```

它指向函数增长最快的方向。

方向导数：

```text
D_u f = ∇f · u
```

其中 `u` 是单位方向向量。

### 20.3 为什么梯度是最陡方向

点积：

```text
∇f · u = |∇f| |u| cos θ
```

如果 `u` 是单位向量：

```text
D_u f = |∇f| cos θ
```

最大值出现在 `θ=0`，也就是 `u` 与梯度同向。

最大方向导数为：

```text
|∇f|
```

### 20.4 梯度与等高线

梯度垂直于等高线。

直觉：

```text
沿等高线走，函数值不变；
变化率为 0；
所以这个方向与梯度点积为 0；
因此梯度垂直等高线切向。
```

这在优化和机器学习中非常重要。

### 20.5 梯度例题：从等高线理解最陡上升

考虑：

```text
f(x,y)=x^2+y^2
```

等高线是：

```text
x^2+y^2=c
```

也就是以原点为中心的圆。

梯度：

```text
∇f=(2x,2y)
```

在点 `(1,2)`：

```text
∇f(1,2)=(2,4)
```

这个向量从原点向外指，正好垂直于经过 `(1,2)` 的圆形等高线。

这很符合直觉：

```text
f=x^2+y^2 表示到原点距离的平方；
增长最快的方向就是远离原点的方向。
```

如果沿单位方向：

```text
u=(1/√5, 2/√5)
```

方向导数为：

```text
∇f · u = (2,4) · (1/√5,2/√5)
       = 10/√5
       = 2√5
```

这等于 `|∇f|`，说明这是最陡方向。

学习梯度时要同时记住三句话：

```text
梯度是偏导数组成的向量。
梯度指向函数增长最快方向。
梯度垂直于等高线或等值面。
```

## 21. 多变量链式法则与隐函数思想

### 21.1 多变量链式法则

如果：

```text
z = f(x,y)
x = x(t)
y = y(t)
```

那么：

```text
dz/dt = ∂f/∂x * dx/dt + ∂f/∂y * dy/dt
```

直觉：

```text
t 改变会通过 x 和 y 两条通道影响 z；
总变化率是两条通道贡献相加。
```

### 21.2 Jacobian 矩阵

对向量函数：

```text
F: R^n -> R^m
```

Jacobian 是所有一阶偏导数组成的矩阵。

它是多变量函数的局部线性近似：

```text
F(a+h) ≈ F(a) + J_F(a) h
```

这和单变量：

```text
f(a+h) ≈ f(a) + f'(a)h
```

完全对应。

### 21.3 隐函数

如果：

```text
F(x,y)=0
```

并且局部可以把 `y` 看成 `x` 的函数，那么：

```text
dy/dx = -F_x / F_y
```

这来自：

```text
F_x + F_y y' = 0
```

前提是 `F_y ≠ 0`。

### 21.4 多变量链式法则的路径图

链式法则最容易在变量关系复杂时出错。建议先画依赖图。

例如：

```text
z = f(x,y)
x = r cos θ
y = r sin θ
```

如果要求 `∂z/∂r`，依赖路径是：

```text
r -> x -> z
r -> y -> z
```

所以：

```text
∂z/∂r = ∂f/∂x * ∂x/∂r + ∂f/∂y * ∂y/∂r
       = f_x cos θ + f_y sin θ
```

如果要求 `∂z/∂θ`，依赖路径是：

```text
θ -> x -> z
θ -> y -> z
```

所以：

```text
∂z/∂θ = f_x (-r sin θ) + f_y (r cos θ)
```

这说明多变量链式法则不是死记公式，而是在依赖图上把所有路径贡献相加。

## 22. 多变量优化：无约束与约束

### 22.1 无约束优化

对 `f(x,y)` 找极值。

候选点满足：

```text
∇f = 0
```

即：

```text
f_x = 0
f_y = 0
```

### 22.2 Hessian 判别

二阶导数组成 Hessian：

```text
H = [[f_xx, f_xy],
     [f_yx, f_yy]]
```

二维常用判别：

```text
D = f_xx f_yy - (f_xy)^2
```

在临界点：

- `D > 0` 且 `f_xx > 0`：局部最小。
- `D > 0` 且 `f_xx < 0`：局部最大。
- `D < 0`：鞍点。
- `D = 0`：无法判断。

### 22.3 拉格朗日乘子

约束优化：

```text
最大/最小 f(x,y)
约束 g(x,y)=c
```

在约束曲线上极值点，梯度满足：

```text
∇f = λ ∇g
```

直觉：

```text
在约束曲线上，如果还能沿约束方向增加 f，就不是极值。
极值处 f 的等高线与约束曲线相切。
所以两个梯度方向平行。
```

### 22.4 解题流程

```text
1. 写目标函数 f。
2. 写约束 g=c。
3. 建方程 ∇f = λ∇g。
4. 加上约束方程。
5. 解候选点。
6. 比较函数值，并检查边界。
```

### 22.5 优化题例子：从梯度到约束

无约束例子：

```text
f(x,y)=x^2+y^2-2x-4y
```

求梯度：

```text
∇f=(2x-2, 2y-4)
```

令梯度为 0：

```text
2x-2=0 -> x=1
2y-4=0 -> y=2
```

Hessian：

```text
[[2,0],[0,2]]
```

正定，所以 `(1,2)` 是局部最小值。事实上，配方可得：

```text
f(x,y)=(x-1)^2+(y-2)^2-5
```

所以它也是全局最小值。

约束例子：

```text
在 x^2+y^2=1 上最大化 f(x,y)=x+y
```

约束圆上，`x+y` 的等高线是斜直线。最大值出现在直线与圆相切处。

用拉格朗日乘子：

```text
∇f=(1,1)
∇g=(2x,2y)
```

方程：

```text
(1,1)=λ(2x,2y)
x^2+y^2=1
```

得到：

```text
x=y
2x^2=1
x=y=1/√2
```

最大值：

```text
√2
```

这个例子要理解：

```text
约束极值不是让 ∇f=0；
而是让 ∇f 没有沿约束曲线继续增加的切向分量。
```

## 23. 重积分：面积、体积、质量与概率

### 23.1 二重积分

```text
∫∫_D f(x,y) dA
```

表示在区域 `D` 上累积密度 `f`。

如果 `f=1`：

```text
∫∫_D 1 dA = 区域面积
```

如果 `f` 是高度：

```text
∫∫_D f(x,y) dA = 曲面下体积
```

如果 `f` 是面密度：

```text
∫∫_D f(x,y) dA = 质量
```

### 23.2 迭代积分

如果区域简单：

```text
∫∫_D f(x,y) dA = ∫_a^b ∫_{g1(x)}^{g2(x)} f(x,y) dy dx
```

关键是画区域。

优秀学生会先问：

```text
D 的边界是什么？
先对 x 积分还是先对 y 积分更简单？
```

### 23.3 积分顺序

有时换顺序能大幅简化。

例如区域：

```text
0 <= x <= 1
x <= y <= 1
```

改写为：

```text
0 <= y <= 1
0 <= x <= y
```

画图比硬背上下限可靠。

### 23.4 三重积分

```text
∫∫∫_E f(x,y,z) dV
```

表示空间区域 `E` 中的累积量。

如果 `f=1`，就是体积。

如果 `f` 是体密度，就是质量。

### 23.5 重积分建模的关键：先画区域

重积分的难点通常不在积分本身，而在区域描述。

例如区域：

```text
D: 0 <= y <= x, 0 <= x <= 1
```

这表示：

```text
x 从 0 到 1；
对每个固定 x，y 从 0 到 x。
```

如果换顺序，需要重新描述同一区域：

```text
0 <= y <= 1
y <= x <= 1
```

很多错误来自把上下限写成记忆模板，而不是从图像读出来。

设置二重积分时，建议每次都写三句话：

```text
外层变量从哪里到哪里？
固定外层变量后，内层变量从哪条边到哪条边？
这个描述是否覆盖区域一次且只覆盖一次？
```

例子：计算三角形区域 `0 <= y <= x <= 1` 的面积。

```text
面积 = ∫_0^1 ∫_0^x 1 dy dx
     = ∫_0^1 x dx
     = 1/2
```

换顺序：

```text
面积 = ∫_0^1 ∫_y^1 1 dx dy
     = ∫_0^1 (1-y) dy
     = 1/2
```

两种方式一致，说明区域描述正确。

## 24. 坐标变换与雅可比行列式

### 24.1 极坐标

```text
x = r cos θ
y = r sin θ
```

面积元：

```text
dA = r dr dθ
```

为什么多了一个 `r`？

因为极坐标中小块近似为：

```text
径向宽度 dr
角向弧长 r dθ
面积 ≈ r dr dθ
```

### 24.2 柱坐标和球坐标

柱坐标：

```text
x = r cos θ
y = r sin θ
z = z
dV = r dr dθ dz
```

球坐标常见形式：

```text
x = ρ sin φ cos θ
y = ρ sin φ sin θ
z = ρ cos φ
dV = ρ^2 sin φ dρ dφ dθ
```

这里 `φ` 表示从正 z 轴量下来的极角。不同教材有时会交换 `θ` 和 `φ` 的命名，使用时要先确认符号约定。

### 24.3 Jacobian 的意义

坐标变换：

```text
(u,v) -> (x,y)
```

面积缩放因子是：

```text
|∂(x,y)/∂(u,v)|
```

直觉：

```text
一个小矩形在变换后变成小平行四边形；
Jacobian 行列式给出面积缩放比例。
```

不要把 Jacobian 当成机械补因子。

它是“局部面积/体积被放大多少”的量。

### 24.4 什么时候换坐标

换坐标不是为了让公式更复杂，而是为了让区域或函数更简单。

优先考虑极坐标的信号：

```text
x^2 + y^2
圆、圆环、扇形
关于原点对称
```

例子：

```text
∫∫_D e^{-(x^2+y^2)} dA
```

如果 `D` 是半径为 `R` 的圆盘，直角坐标下边界麻烦，但极坐标中：

```text
x^2+y^2 = r^2
dA = r dr dθ
0 <= r <= R
0 <= θ <= 2π
```

积分变成：

```text
∫_0^{2π} ∫_0^R e^{-r^2} r dr dθ
```

内层可以换元：

```text
u = r^2
du = 2r dr
```

这说明坐标变换同时简化了区域和被积函数。

使用 Jacobian 时要检查：

```text
变换是否一对一或是否正确分区？
新变量范围是什么？
面积元或体积元是否已经乘上缩放因子？
```

## 25. 向量场、线积分与面积分

### 25.1 向量场

向量场给空间中每个点分配一个向量：

```text
F(x,y) = (P(x,y), Q(x,y))
```

例子：

- 风速场。
- 流体速度场。
- 电场。
- 力场。

### 25.2 标量线积分

沿曲线 `C` 积分标量函数：

```text
∫_C f ds
```

表示沿路径累积密度。

如果 `f` 是线密度，积分就是质量。

### 25.3 向量线积分

```text
∫_C F · dr
```

表示向量场沿路径方向做的功。

如果路径参数化：

```text
r(t), a <= t <= b
```

那么：

```text
∫_C F · dr = ∫_a^b F(r(t)) · r'(t) dt
```

### 25.4 保守场

如果存在势函数 `φ`：

```text
F = ∇φ
```

则 `F` 是保守场。

线积分只与端点有关：

```text
∫_C F · dr = φ(B) - φ(A)
```

这类似一维基本定理的多维版本。

### 25.5 面积分

标量面积分：

```text
∫∫_S f dS
```

向量通量积分：

```text
∫∫_S F · n dS
```

通量表示向量场穿过曲面的总量。

### 25.6 如何区分几种积分

向量微积分里最容易混淆的是：

```text
∫ f dx
∫_C f ds
∫_C F · dr
∫∫_S f dS
∫∫_S F · n dS
```

它们对应的问题不同：

| 积分 | 小量 | 典型问题 |
| --- | --- | --- |
| 单变量积分 | `f(x) dx` | 沿 x 轴累积 |
| 标量线积分 | `f ds` | 沿曲线按弧长累积质量 |
| 向量线积分 | `F · dr` | 力沿路径做功 |
| 标量面积分 | `f dS` | 曲面上累积密度 |
| 通量积分 | `F · n dS` | 穿过曲面的流量 |

判断方法：

```text
如果题目说沿路径累积密度，用 f ds。
如果题目说力场做功，用 F · dr。
如果题目说穿过曲面，用 F · n dS。
```

例子：

```text
F=(y,x), 路径 r(t)=(t,t^2), 0<=t<=1
```

向量线积分：

```text
∫_C F · dr = ∫_0^1 F(r(t)) · r'(t) dt
```

代入：

```text
F(r(t))=(t^2,t)
r'(t)=(1,2t)
点积 = t^2 + 2t^2 = 3t^2
```

所以：

```text
∫_0^1 3t^2 dt = 1
```

这道题的关键不是公式，而是把路径参数化后，沿路径方向取向量场分量。

## 26. Green、Stokes 与散度定理

这些定理看起来复杂，但共享一个思想：

```text
边界上的累积量 = 区域内部某种变化的累积量
```

### 26.1 Green 定理

平面区域 `D` 的边界为 `C`，并且 `C` 按逆时针正向绕行：

```text
∮_C P dx + Q dy = ∫∫_D (∂Q/∂x - ∂P/∂y) dA
```

它把边界曲线积分转化为区域二重积分。

直觉：

```text
区域内部微小旋转的总和，等于边界上的环流。
```

### 26.2 Stokes 定理

```text
∮_C F · dr = ∫∫_S (curl F) · n dS
```

它是 Green 定理在三维曲面上的推广。

这里曲面法向量 `n` 与边界曲线 `C` 的方向必须匹配右手定则。方向反了，积分符号会变号。

直觉：

```text
曲面上每个小块的旋转累积起来，等于边界曲线上的总环流。
```

### 26.3 散度定理

```text
∫∫_S F · n dS = ∫∫∫_E div F dV
```

这里 `S` 是空间区域 `E` 的封闭边界曲面，`n` 取外法向量。

直觉：

```text
区域内部源头产生的总流量，等于穿出边界的总通量。
```

### 26.4 统一理解

这些定理都体现：

```text
局部变化 -> 整体边界效应
```

与一维基本定理对应：

```text
∫_a^b f'(x) dx = f(b) - f(a)
```

一维中，边界是两个端点。

二维中，边界是闭合曲线。

三维中，边界是曲面。

### 26.5 三大定理的解题判断

看到 Green、Stokes 或散度定理，不要先背公式，而要判断“边界和内部哪个更容易”。

Green 定理适合：

```text
平面闭合曲线积分难算；
区域二重积分更简单。
```

Stokes 定理适合：

```text
空间闭合曲线积分难算；
找一个边界相同但更简单的曲面来算旋度通量。
```

散度定理适合：

```text
封闭曲面通量难算；
体积分更简单。
```

统一模板：

```text
1. 识别是边界积分还是区域/曲面积分。
2. 检查方向约定。
3. 判断是否闭合。
4. 计算内部的导数对象：旋度或散度。
5. 选择更简单的一边计算。
```

例子：平面上单位圆 `C` 逆时针，计算：

```text
∮_C -y dx + x dy
```

这里：

```text
P=-y
Q=x
```

Green 定理给出：

```text
∂Q/∂x - ∂P/∂y = 1 - (-1) = 2
```

单位圆面积为 `π`，所以：

```text
∮_C -y dx + x dy = ∫∫_D 2 dA = 2π
```

直觉：

```text
这个向量场绕原点旋转，单位圆上的总环流与圆盘面积成正比。
```

## 27. 如何像优秀学生一样解题

### 27.1 先分类问题

看到题目，先判断类型：

```text
极限？
连续？
求导？
优化？
积分？
级数？
多变量？
向量场？
建模？
```

不要急着套公式。

### 27.2 写出对象和条件

优秀解题过程通常先写：

```text
函数是什么？
变量范围是什么？
需要证明或计算什么？
已知条件有什么限制？
```

例如：

```text
在闭区间上找最大值
```

要立刻想到：

```text
检查连续性；
找临界点；
检查端点。
```

### 27.3 画图和估计

图像不只是辅助。

它能帮你发现：

- 极限可能不存在。
- 积分区域上下限写反。
- 函数是否有对称性。
- 极值在边界还是内部。
- 级数项是否正负交替。

### 27.4 先做粗略判断

计算前先估计答案特征：

```text
结果应该为正还是负？
量纲是否正确？
面积是否可能超过矩形外界？
概率是否在 0 和 1 之间？
极限是否应为有限数？
```

这能及时发现计算错误。

### 27.5 用反例训练严谨性

每学一个定理，都问：

```text
如果去掉某个条件，结论还成立吗？
```

例如：

```text
可导 => 连续，但连续不一定可导。
级数项趋于 0，不代表级数收敛。
偏导存在，不一定函数连续。
闭区间连续函数有最大最小值，开区间不一定。
```

反例意识是从做题者走向理解者的重要一步。

### 27.6 一份完整解题模板

遇到陌生题，可以照下面写草稿：

```text
题目类型：
已知条件：
要求目标：
相关定义：
可用定理：
需要检查的条件：
计算主线：
结果合理性检查：
```

例如优化题：

```text
题目类型：单变量优化
已知条件：变量范围、约束关系
要求目标：最大/最小某个量
相关定义：目标函数
可用定理：导数临界点、端点检查
需要检查的条件：定义域、闭区间、边界
计算主线：建模 -> 求导 -> 候选点 -> 比较
结果检查：量纲、符号、边界是否合理
```

例如级数题：

```text
题目类型：收敛性判断
已知条件：通项 a_n
要求目标：收敛、发散、绝对收敛或条件收敛
相关定义：部分和、绝对收敛
可用定理：比值、根值、比较、交错
需要检查的条件：正项？递减？趋于 0？
计算主线：先看 a_n 是否趋于 0，再选判别法
结果检查：是否与增长速度直觉一致
```

这种模板能帮助你避免“看见题就乱套公式”。

## 28. 常见误区与反例意识

### 28.1 误区：极限就是代入

代入只是某些连续函数极限的快捷方法。

遇到未定式，需要分析结构。

### 28.2 误区：导数为 0 就一定是极值

反例：

```text
f(x)=x^3
```

在 0 导数为 0，但不是极值。

### 28.3 误区：函数连续就一定可导

反例：

```text
f(x)=|x|
```

在 0 连续但不可导。

### 28.4 误区：积分结果一定是面积

定积分可以是有符号面积。

如果函数在 x 轴下方，积分贡献为负。

求几何面积时要积分绝对值或分段处理。

### 28.5 误区：无穷小就是普通数字 0

微积分中的小量思想很有用，但严谨计算依赖极限。

不能随意把 `dx` 当普通数字乱消。

### 28.6 误区：级数项趋于 0 就收敛

反例：

```text
Σ 1/n
```

项趋于 0，但级数发散。

### 28.7 误区：偏导存在就连续

多变量中，偏导存在不一定连续，甚至不一定函数连续。

多变量分析比单变量更需要路径和局部线性化意识。

## 29. 学习路线与练习计划

### 29.1 第一阶段：函数、极限、连续

目标：

- 会判断左右极限。
- 会处理常见未定式。
- 理解连续的三个条件。
- 能构造简单反例。

练习：

1. 计算 20 个基础极限。
2. 找 5 个极限不存在的例子。
3. 画出可去、跳跃、无穷不连续的图像。

### 29.2 第二阶段：导数和应用

目标：

- 熟练使用求导规则。
- 会用导数分析单调性、凹凸性和极值。
- 会建模简单优化题。

练习：

1. 用定义求 `x^2`、`1/x`、`sqrt(x)` 的导数。
2. 做 30 个链式法则题。
3. 做 10 个优化建模题。

### 29.3 第三阶段：积分和应用

目标：

- 理解积分是累积。
- 掌握换元、分部积分、部分分式。
- 会用积分求面积、体积、功、概率。

练习：

1. 每种积分技巧做 15 题。
2. 画区域并设置二重积分前的单变量面积题。
3. 比较圆盘法和壳层法。

### 29.4 第四阶段：级数和 Taylor

目标：

- 会判断常见级数收敛。
- 理解幂级数和收敛半径。
- 会用 Taylor 做近似和误差估计。

练习：

1. 对 30 个级数选择判别法。
2. 推导 `e^x`、`sin x`、`cos x` 的 Taylor 展开。
3. 用 Taylor 估算 `e^0.1` 和 `sin(0.1)`。

### 29.5 第五阶段：多变量和向量微积分

目标：

- 理解偏导、梯度、方向导数。
- 会做二重积分和坐标变换。
- 理解线积分、面积分和三大定理。

练习：

1. 画 10 个二重积分区域并交换积分顺序。
2. 计算 10 个梯度和方向导数。
3. 用 Green 定理计算平面闭合曲线积分。

### 29.6 每周学习节奏

建议：

```text
20% 时间看概念和图像
30% 时间做基础计算
30% 时间做应用题
20% 时间复盘错题和反例
```

不要只刷计算题。

微积分真正的能力来自：

```text
能把一个陌生问题翻译成变化率、累积量、近似、极值或空间流动问题。
```
