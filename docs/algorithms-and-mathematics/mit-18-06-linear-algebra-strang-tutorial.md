## 0. Strang 的整体思路：矩阵不是数字表

线性代数最容易学坏的方式，是把它看成一组互不相干的计算：今天消元，明天行列式，后天特征值。Strang 的教学顺序反复强调另一件事：**先抓住矩阵所表达的结构，再把计算理解为揭示结构的工具。**

一个矩阵 `A` 至少可以同时读成五种对象：

| 视角 | `A` 表示什么 | 典型问题 |
| --- | --- | --- |
| 方程视角 | 线性方程组 `Ax=b` 的系数 | 有解吗？解唯一吗？ |
| 列视角 | 一组列向量 | `b` 能否由这些列组合出来？ |
| 行视角 | 一组线性约束 | 多个超平面在哪里相交？ |
| 变换视角 | 从输入空间到输出空间的线性映射 | 哪些方向被压缩、拉伸或消灭？ |
| 分解视角 | 若干简单变换的乘积 | 能否换到更容易计算的坐标？ |

![MIT 18.06 线性代数的大图景](https://oss.euler.icu/teaser/docs/algorithms-and-mathematics/linear-algebra-18-06/01-big-picture.png?v=20260711-final)

### 0.1 贯穿整门课的四个问题

遇到一个新矩阵，先不要急着套公式。先问：

```text
1. 存在性：Ax=b 是否有解？
2. 唯一性：解是一个点、一个仿射子空间，还是不存在？
3. 几何性：A 对空间中的方向做了什么？
4. 稳定性：数据稍微变化，答案会不会剧烈变化？
```

这四问分别把我们带向：

- 列空间与左零空间；
- 零空间与秩；
- 特征向量与 SVD；
- 奇异值与条件数。

### 0.2 计算不是目的，而是观察仪器

Strang 式的计算都在回答结构问题：

| 计算 | 暴露的结构 |
| --- | --- |
| 消元得到 `R` 或 `U` | 主元位置、秩、自由变量、可解性 |
| `A=LU` | 消元过程可以复用 |
| `A=QR` | 列空间的一组正交坐标 |
| `A=XΛX⁻¹` | 变换的不变方向与长期行为 |
| `A=UΣVᵀ` | 任意矩阵的输入方向、强度和输出方向 |

学习每个算法时都要追问：**输出矩阵里的每一块，究竟记录了什么？**

## 1. 向量：线性代数的基本对象

### 1.1 一个向量的四种解释

向量 `v=(v₁,...,vₙ)ᵀ` 可以是：

- 从原点出发的有方向线段；
- 空间中的一个点；
- 一组特征或测量值；
- 某组基向量的坐标。

例如 `v=(2,-1)ᵀ` 既可以表示平面中的位移，也可以表示：

```text
v = 2e₁ - e₂
```

这里 `e₁=(1,0)ᵀ`、`e₂=(0,1)ᵀ` 是标准基。坐标本身不是向量的全部意义；它们依赖于所选的基。

### 1.2 线性组合是第一核心动作

给定 `v₁,...,vₖ`，表达式

```text
c₁v₁ + ··· + cₖvₖ
```

称为线性组合。线性代数中的大量问题都可翻译成：

```text
目标向量能否由给定向量线性组合出来？
如果能，系数是否唯一？
```

这正是 `Ax=b` 的列图像，因为若

```text
A=[a₁ a₂ ... aₙ],  x=(x₁,...,xₙ)ᵀ
```

那么

```text
Ax = x₁a₁ + ··· + xₙaₙ。
```

### 1.3 点积、长度和角度

实向量的点积为

```text
vᵀw = Σᵢ vᵢwᵢ。
```

它同时编码长度与角度：

```text
||v||₂ = sqrt(vᵀv)
vᵀw = ||v||₂ ||w||₂ cos θ。
```

因此：

- `vᵀw=0` 表示正交；
- `|vᵀw| ≤ ||v|| ||w||` 是 Cauchy-Schwarz 不等式；
- `||v+w|| ≤ ||v||+||w||` 是三角不等式。

点积不是只为“算角度”。后面的投影、最小二乘、正交基、Fourier 系数都来自同一个动作：**用点积测量一个向量在某方向上的分量。**

### 1.4 一个小例子

令 `v=(3,4)ᵀ`，`q=(1,0)ᵀ`。则：

```text
||v||=5
qᵀv=3
v 在 q 方向的投影向量 = (qᵀv)q = (3,0)ᵀ。
```

若 `q` 不是单位向量，投影系数要除以 `qᵀq`。这一区别会在最小二乘中反复出现。

## 2. 矩阵乘法：同时看列、行和变换

### 2.1 `Ax` 的列组合解释

这是最重要的矩阵乘法解释：

```text
Ax = x₁a₁ + ··· + xₙaₙ。
```

它立即说明：所有可能的 `Ax` 组成 `A` 的列空间。

### 2.2 `AB` 的四种读法

若 `A` 为 `m×n`，`B` 为 `n×p`，则 `AB` 为 `m×p`。可以这样读：

1. **逐元素**：`(AB)ᵢⱼ` 是 `A` 第 `i` 行与 `B` 第 `j` 列的点积。
2. **逐列**：`AB` 的第 `j` 列是 `A` 乘 `B` 的第 `j` 列。
3. **逐行**：`AB` 的第 `i` 行是 `A` 第 `i` 行乘 `B`。
4. **外积和**：若 `A=[a₁ ... aₙ]`，`B` 的各行为 `r₁ᵀ,...,rₙᵀ`，则

```text
AB = a₁r₁ᵀ + ··· + aₙrₙᵀ。
```

最后一种读法直接通向秩一矩阵与 SVD。

### 2.3 为什么矩阵乘法一般不可交换

`AB` 表示先执行 `B`，再执行 `A`。操作次序改变，结果通常改变：

```text
AB ≠ BA。
```

例如“先旋转再投影”与“先投影再旋转”通常不同。应把不可交换理解为变换顺序的差异，而不是孤立规则。

### 2.4 结合律为什么成立

```text
(AB)C = A(BC)
```

因为两边都表示依次执行 `C`、`B`、`A`。结合律允许我们选择计算代价较低的括号顺序，但不能交换因子顺序。

### 2.5 块矩阵

当分块尺寸兼容时，块矩阵按普通矩阵规则相乘。例如：

```text
[A B] [x] = Ax + By。
      [y]
```

块矩阵不是排版技巧。它表达“系统由哪些子系统耦合而成”，在图网络、最优化、数值算法中极其重要。

## 3. `Ax=b` 的行图像与列图像

线性方程组是 18.06 的入口。最关键的不是更快地解两个方程，而是学会在两个几何世界之间切换。

![Ax=b 的行图像与列图像](https://oss.euler.icu/teaser/docs/algorithms-and-mathematics/linear-algebra-18-06/02-row-column-pictures.png?v=20260711-final)

考虑：

```text
x + 2y = 5
2x - y = 0。
```

写成：

```text
[1  2] [x] = [5]
[2 -1] [y]   [0]。
```

### 3.1 行图像：多个约束的交集

每一行是一个方程，在二维中是一条直线。求解就是寻找两条直线的交点 `(1,2)`。

在三维中，一行通常表示一个平面；在 `n` 维中，它表示一个超平面。行图像直观，但维数较高时难以画出。

### 3.2 列图像：寻找生成 `b` 的组合

同一方程也可以写成：

```text
x(1,2)ᵀ + y(2,-1)ᵀ = (5,0)ᵀ。
```

所以问题变成：

```text
b 是否位于 A 的列向量张成的空间中？
```

这里 `b=1a₁+2a₂`，所以 `x=1,y=2`。

### 3.3 存在性与唯一性已经出现

- `b∈C(A)` 当且仅当 `Ax=b` 有解；
- 若列向量线性无关，组合系数至多一组；
- 若列向量相关且有解，通常会有自由变量。

后面的列空间、零空间、秩，本质上是在把这三句话精确化。

## 4. 消元、主元与 `PA=LU`

### 4.1 消元在做什么

Gaussian 消元通过行操作把矩阵变为阶梯形：

```text
用第一个主元消去它下方的元素；
再用第二个主元消去它下方的元素；
直到得到上三角矩阵 U。
```

允许的三种行操作是：

1. 交换两行；
2. 某行乘非零数；
3. 某行加上另一行的倍数。

它们都不改变方程组的解集，因为每一步都可以逆转。

![消元与 PA=LU](https://oss.euler.icu/teaser/docs/algorithms-and-mathematics/linear-algebra-18-06/03-pa-lu-elimination.png?v=20260711-final)

### 4.2 主元是结构信号

主元不仅是计算中“拿来除的数”。主元位置告诉我们：

- 有多少个独立约束；
- 哪些变量是主变量；
- 哪些变量自由；
- 矩阵的秩是多少；
- 方阵是否可逆。

如果消元遇到零主元，但下方有非零元素，需要交换行。用置换矩阵 `P` 记录交换后，一般形式是：

```text
PA = LU。
```

### 4.3 一个完整 `LU` 例子

令

```text
A = [ 2  1  1]
    [ 4 -6  0]
    [-2  7  2]。
```

消元乘子为：

```text
ℓ₂₁=2，ℓ₃₁=-1，ℓ₃₂=-1。
```

得到：

```text
L = [ 1  0  0]     U = [2  1  1]
    [ 2  1  0]         [0 -8 -2]
    [-1 -1  1]         [0  0  1]

A=LU。
```

检查时不要重新做消元，直接计算 `LU` 是否回到 `A`。

### 4.4 为什么分解比一次求解更重要

若要对多个右端项求解 `Ax=b₁,Ax=b₂,...`，只需分解一次：

```text
Ly=b      前代
Ux=y      回代。
```

稠密 `n×n` 矩阵的 LU 分解约需 `O(n³)` 运算，而每个新右端项只需 `O(n²)`。这就是算法设计中的“预处理一次，重复查询便宜”。

### 4.5 数值计算为什么要选主元

数学上非零的主元，如果绝对值极小，也会放大舍入误差。数值软件通常使用部分选主元：在当前列剩余元素中选择绝对值最大的元素作为主元。

因此：

```text
纯理论常写 A=LU；
可靠实现通常计算 PA=LU。
```

## 5. 逆矩阵与可逆矩阵定理

### 5.1 逆矩阵的含义

方阵 `A` 若存在 `A⁻¹` 使

```text
A⁻¹A=AA⁻¹=I，
```

则 `A` 可逆。变换视角下，`A⁻¹` 撤销 `A` 的作用。

### 5.2 计算逆矩阵

Gauss-Jordan 方法对增广矩阵消元：

```text
[A | I]  →  [I | A⁻¹]。
```

这等价于一次求解 `n` 个方程：

```text
Ax₁=e₁, ..., Axₙ=eₙ。
```

### 5.3 为什么求 `Ax=b` 通常不显式计算逆

表达式 `x=A⁻¹b` 很简洁，但数值实现通常直接使用 LU、QR 或其他分解，因为：

- 显式逆需要更多运算和存储；
- 可能引入更多舍入误差；
- 分解能利用稀疏、对称、正定等结构。

数学表达与计算算法应当区分。

### 5.4 可逆矩阵定理的核心等价关系

对 `n×n` 方阵 `A`，以下陈述等价：

```text
A 可逆
⇔ 每列都有主元
⇔ rank(A)=n
⇔ N(A)={0}
⇔ 列向量线性无关
⇔ 列空间 C(A)=Rⁿ
⇔ 对每个 b，Ax=b 有唯一解
⇔ det(A)≠0
⇔ 0 不是 A 的特征值。
```

不要把它背成九条孤立结论。逻辑链是：

```text
没有自由方向
→ 不会把两个不同输入映到同一输出
→ 每个输出恰有一个原像
→ 变换可以撤销。
```

## 6. 向量空间与子空间

### 6.1 为什么需要“空间”

求一个方程的答案可能是一个向量；求所有可能答案，往往得到一整组向量。为了研究这组向量是否对线性组合封闭，我们引入向量空间。

### 6.2 子空间判定

集合 `S⊆Rⁿ` 是子空间，当且仅当：

```text
0∈S；
u,v∈S ⇒ u+v∈S；
u∈S,c∈R ⇒ cu∈S。
```

后两条也可合并为：对任意 `u,v∈S` 和标量 `α,β`，有 `αu+βv∈S`。

### 6.3 常见反例

- 过原点的直线是子空间；不过原点的平行直线不是。
- 平面 `x+y+z=0` 是子空间；`x+y+z=1` 不是。
- 所有次数不超过 `k` 的多项式构成子空间。
- 所有可逆矩阵不构成子空间，因为两个可逆矩阵之和可能不可逆，且零矩阵不在其中。

### 6.4 张成

```text
span{v₁,...,vₖ}
```

是包含这些向量的最小子空间，也就是它们全部线性组合的集合。列空间正是矩阵各列的张成。

## 7. 零空间：求解 `Ax=0`

### 7.1 零空间是什么

```text
N(A)={x∈Rⁿ:Ax=0}。
```

它收集所有被 `A` 压到零向量的输入方向。零空间一定是子空间，因为线性性保证：

```text
Au=0, Av=0
⇒ A(αu+βv)=αAu+βAv=0。
```

### 7.2 用 RREF 求零空间

例：

```text
A = [1 2 3]
    [2 4 6]。
```

消元得到：

```text
R = [1 2 3]
    [0 0 0]。
```

`x₁` 是主变量，`x₂=s,x₃=t` 是自由变量：

```text
x₁=-2s-3t

x = s(-2,1,0)ᵀ + t(-3,0,1)ᵀ。
```

所以：

```text
N(A)=span{(-2,1,0)ᵀ,(-3,0,1)ᵀ}。
```

### 7.3 特殊解为什么构成基

每次把一个自由变量设为 1、其余自由变量设为 0，就得到一个特殊解。自由变量有 `n-r` 个，因此：

```text
dim N(A)=n-r。
```

这就是秩-零度定理的一半。

### 7.4 易错点

- 行操作不改变零空间方程的解，因此可从 `R` 求 `N(A)`。
- 但行操作会改变列向量本身，不能直接拿 `R` 的主元列作为 `C(A)` 的基；要回到原矩阵 `A` 取对应列。

## 8. 列空间：判断 `Ax=b` 是否有解

### 8.1 定义

```text
C(A)={Ax:x∈Rⁿ}=span{A 的各列}⊆Rᵐ。
```

于是最重要的存在性判据是：

```text
Ax=b 有解 ⇔ b∈C(A)。
```

### 8.2 如何求列空间的一组基

仍用：

```text
A = [1 2 3]
    [2 4 6]。
```

RREF 只有第一列为主元列，因此回到原矩阵取第一列：

```text
C(A)=span{(1,2)ᵀ}。
```

第二列是第一列的 2 倍，第三列是 3 倍，没有新增方向。

### 8.3 增广矩阵怎样暴露无解

若对 `[A|b]` 消元出现：

```text
[0 0 ... 0 | c]，c≠0，
```

就得到矛盾 `0=c`，说明 `b∉C(A)`。

这把代数矛盾与几何事实连接起来：目标向量落在列空间之外。

## 9. 线性无关、基、维数与秩

### 9.1 线性无关

向量 `v₁,...,vₖ` 线性无关，意思是：

```text
c₁v₁+···+cₖvₖ=0
```

只有零系数解。若存在非零系数组合为零，则至少一个向量可以由其他向量表示，它没有提供新方向。

### 9.2 基必须同时满足两件事

空间 `S` 的一组基是：

```text
既能张成 S，又线性无关的一组向量。
```

“能张成”防止缺方向，“线性无关”防止冗余。

### 9.3 维数

有限维空间的任何基都含相同数量的向量，这个数量称为维数。维数不是坐标长度，而是独立方向的数量。

### 9.4 秩

矩阵的秩 `r` 是主元个数，也是：

```text
r = dim C(A) = dim C(Aᵀ)。
```

这说明列秩等于行秩。消元之所以强大，是因为同一组主元同时暴露了输入侧与输出侧的独立方向数。

### 9.5 秩-零度定理

对 `A∈R^{m×n}`：

```text
rank(A)+nullity(A)=n
r+dim N(A)=n。
```

输入空间的 `n` 个方向被分成：

- `r` 个真正传到输出空间的行空间方向；
- `n-r` 个被压成零的零空间方向。

## 10. 四个基本子空间

这是 Strang 课程的中心图景之一。对 `A∈R^{m×n}`、`rank(A)=r`：

![四个基本子空间](https://oss.euler.icu/teaser/docs/algorithms-and-mathematics/linear-algebra-18-06/04-four-fundamental-subspaces.png?v=20260711-final)

| 子空间 | 所在空间 | 维数 | 计算方式 |
| --- | --- | --- | --- |
| 列空间 `C(A)` | `Rᵐ` | `r` | 原矩阵的主元列 |
| 左零空间 `N(Aᵀ)` | `Rᵐ` | `m-r` | 解 `Aᵀy=0` |
| 行空间 `C(Aᵀ)` | `Rⁿ` | `r` | RREF 的非零行 |
| 零空间 `N(A)` | `Rⁿ` | `n-r` | 解 `Ax=0` |

### 10.1 两组正交补

```text
C(Aᵀ)⊥N(A)
C(A)⊥N(Aᵀ)。
```

证明第一条：若 `x∈N(A)`，则 `Ax=0`。`A` 的每一行 `rᵢᵀ` 都满足 `rᵢᵀx=0`，所以 `x` 与每个行向量正交，也与整个行空间正交。

第二条同理，或把第一条应用于 `Aᵀ`。

### 10.2 直和分解

每个输入向量都可唯一写成：

```text
x=x_r+x_n，
x_r∈C(Aᵀ)，x_n∈N(A)。
```

由于 `Ax_n=0`，有：

```text
Ax=Ax_r。
```

矩阵只“看见”输入的行空间部分，完全看不见零空间部分。

输出空间也分解为：

```text
Rᵐ=C(A)⊕N(Aᵀ)。
```

最小二乘就是把 `b` 分成列空间分量 `p` 和左零空间分量 `e`。

### 10.3 用前面的矩阵算四个空间

对

```text
A=[1 2 3; 2 4 6]，r=1：
```

```text
C(A)=span{(1,2)ᵀ}
N(Aᵀ)=span{(-2,1)ᵀ}
C(Aᵀ)=span{(1,2,3)ᵀ}
N(A)=span{(-2,1,0)ᵀ,(-3,0,1)ᵀ}。
```

检查维数：`1+(2-1)=2`，`1+(3-1)=3`；检查正交：

```text
(1,2)·(-2,1)=0
(1,2,3)·(-2,1,0)=0
(1,2,3)·(-3,0,1)=0。
```

## 11. `Ax=b` 的完整解

![Ax=b 的完整解](https://oss.euler.icu/teaser/docs/algorithms-and-mathematics/linear-algebra-18-06/05-complete-solution.png?v=20260711-final)

### 11.1 为什么是“特解 + 零空间”

假设 `x_p` 是 `Ax=b` 的一个特解，`x_n∈N(A)`。则：

```text
A(x_p+x_n)=Ax_p+Ax_n=b+0=b。
```

反过来，若 `x` 也是解，则：

```text
A(x-x_p)=Ax-Ax_p=b-b=0，
```

所以 `x-x_p∈N(A)`。因此全部解恰好是：

```text
x=x_p+x_n，x_n∈N(A)。
```

### 11.2 图中例子

RREF 增广矩阵为：

```text
[1 2 0 | 4]
[0 0 1 | 1]。
```

令自由变量 `x₂=t`：

```text
x₁=4-2t，x₃=1
x=(4,0,1)ᵀ+t(-2,1,0)ᵀ。
```

特解 `(4,0,1)ᵀ` 把过原点的零空间直线平移成完整解集。完整解集通常不是子空间，因为它未必经过原点；它是仿射子空间。

### 11.3 三种可能性

| 条件 | 解的情况 |
| --- | --- |
| `b∉C(A)` | 无解 |
| `b∈C(A)` 且 `N(A)={0}` | 唯一解 |
| `b∈C(A)` 且 `dim N(A)>0` | 无穷多解 |

这张表比记忆“方程数与未知数数目”更可靠。方程多不代表无解，未知数多也不自动代表无穷多解；关键是秩、列空间和零空间。

## 12. 线性变换、矩阵与换基

### 12.1 线性变换的定义

映射 `T:V→W` 若满足：

```text
T(αu+βv)=αT(u)+βT(v)，
```

就是线性变换。它必须保持线性组合，因此必有 `T(0)=0`。

旋转、反射、投影、缩放、求导都是线性变换；平移 `T(x)=Ax+b` 在 `b≠0` 时不是线性变换，而是仿射变换。

### 12.2 矩阵的列从哪里来

给定输入空间的一组基 `v₁,...,vₙ` 和输出空间的一组基 `w₁,...,wₘ`，只要知道每个基向量被送到哪里，就知道整个变换：

```text
T(vⱼ)=a₁ⱼw₁+···+aₘⱼwₘ。
```

系数 `(a₁ⱼ,...,aₘⱼ)ᵀ` 就是矩阵第 `j` 列。矩阵列是**输入基向量的像在输出基下的坐标**。

标准基下：

```text
A=[T(e₁) T(e₂) ... T(eₙ)]。
```

### 12.3 核与像

线性变换语言下：

```text
ker(T)=N(A)
image(T)=C(A)。
```

秩-零度定理因此可读为：

```text
输入维数 = 真正传出去的方向数 + 被消灭的方向数。
```

### 12.4 复合对应矩阵乘法

若 `T(x)=Ax`、`S(x)=Bx`，则：

```text
(T∘S)(x)=A(Bx)=(AB)x。
```

所以 `AB` 的右因子先执行。这也是矩阵乘法不可交换的根源。

### 12.5 换基不是改变变换

设 `S=[v₁ ... vₙ]` 的列是新基在旧坐标中的表示。若 `x_old=Sx_new`，则同一个线性变换在新基下的矩阵是：

```text
A_new=S⁻¹A_oldS。
```

右边的 `S` 把新坐标转成旧坐标；中间 `A_old` 执行变换；左边 `S⁻¹` 把结果转回新坐标。

换基时最常见的错误，是只乘一侧。记住它是“进去、变换、出来”的三步过程。

## 13. 正交：把耦合问题拆成独立方向

### 13.1 正交为什么特殊

一组两两正交的非零向量必线性无关。证明很直接：若

```text
c₁v₁+···+cₖvₖ=0，
```

两边与 `vⱼ` 点积，其他项全部消失：

```text
cⱼ||vⱼ||²=0 ⇒ cⱼ=0。
```

正交把一个耦合的向量方程拆成多个互不干扰的标量方程。

### 13.2 标准正交基

若向量还满足 `||qᵢ||=1`，则称为标准正交组：

```text
qᵢᵀqⱼ = 1，i=j
qᵢᵀqⱼ = 0，i≠j。
```

把它们作为列组成 `Q`：

```text
QᵀQ=I。
```

若 `Q` 是方阵，则 `QQᵀ=I` 且：

```text
Q⁻¹=Qᵀ。
```

### 13.3 正交矩阵保持长度和角度

```text
||Qx||²=(Qx)ᵀ(Qx)=xᵀQᵀQx=xᵀx。
```

因此正交变换只是旋转或反射，不改变欧氏距离。数值算法偏爱正交变换，因为它不会人为放大误差。

### 13.4 正交子空间

两个子空间 `V,W` 正交，是指任意 `v∈V,w∈W` 都有 `vᵀw=0`。四个基本子空间形成两对正交补，这一结构将直接产生投影和最小二乘。

## 14. 投影：寻找子空间中的最近点

### 14.1 投影到一条直线

要把 `b` 投影到 `a` 张成的直线，令 `p=xa`，并要求误差 `e=b-p` 与 `a` 正交：

```text
aᵀ(b-xa)=0。
```

解得：

```text
x̂=(aᵀb)/(aᵀa)
p=a(aᵀb)/(aᵀa)
P=a(aᵀa)⁻¹aᵀ。
```

若 `a` 是单位向量 `q`：

```text
p=(qᵀb)q，P=qqᵀ。
```

### 14.2 投影到列空间

若 `A` 的列线性无关，希望 `p=A x̂` 是 `b` 在 `C(A)` 上的投影。误差必须与列空间的每一列正交：

```text
Aᵀ(b-Ax̂)=0。
```

得到正规方程：

```text
AᵀA x̂=Aᵀb。
```

若 `A` 满列秩：

```text
x̂=(AᵀA)⁻¹Aᵀb
P=A(AᵀA)⁻¹Aᵀ。
```

### 14.3 投影矩阵的两个身份证

正交投影矩阵满足：

```text
P²=P       再投一次不变
Pᵀ=P       投影方向与目标子空间正交。
```

反过来，实矩阵若同时对称且幂等，就是某个子空间上的正交投影。

### 14.4 `I-P` 投到哪里

```text
b=Pb+(I-P)b=p+e。
```

其中 `p∈C(A)`，`e∈N(Aᵀ)`。这正是输出空间的正交直和：

```text
Rᵐ=C(A)⊕N(Aᵀ)。
```

## 15. 最小二乘：无解方程的最佳近似

当数据含噪声或方程多于未知数时，`Ax=b` 常常无解。最小二乘不假装方程有精确解，而是求：

```text
min_x ||b-Ax||₂²。
```

![正交投影与最小二乘](https://oss.euler.icu/teaser/docs/algorithms-and-mathematics/linear-algebra-18-06/06-projection-least-squares.png?v=20260711-final)

### 15.1 正规方程从哪里来

最佳点 `p=A x̂` 是 `b` 在列空间上的正交投影，所以残差：

```text
e=b-Ax̂
```

必须满足：

```text
Aᵀe=0。
```

展开就是：

```text
AᵀA x̂=Aᵀb。
```

这不是需要背诵的技巧，而是“最近点的误差垂直于可移动方向”。

### 15.2 从微分也能得到同一结果

令：

```text
f(x)=||b-Ax||²=(b-Ax)ᵀ(b-Ax)。
```

则：

```text
∇f(x)=2Aᵀ(Ax-b)。
```

令梯度为零，同样得到正规方程。这连接了线性代数与优化。

### 15.3 拟合一条直线

对数据点 `(0,1),(1,2),(2,2)` 拟合 `y=c+dt`：

```text
A = [1 0]   b=[1]
    [1 1]     [2]
    [1 2]     [2]。
```

正规方程为：

```text
[3 3][c] = [5]
[3 5][d]   [6]。
```

解得：

```text
c=7/6，d=1/2。
```

拟合线：

```text
y=7/6+t/2。
```

残差为：

```text
e=(-1/6,1/3,-1/6)ᵀ。
```

检查：

```text
[1,1,1]·e=0
[0,1,2]·e=0。
```

残差确实与 `A` 的两列都正交。

### 15.4 正规方程的数值风险

理论推导常写 `AᵀA`，但数值上它会把条件数平方：

```text
κ₂(AᵀA)=κ₂(A)²。
```

因此可靠实现通常：

- 满列秩且条件较好：使用 QR；
- 可能秩亏或病态：使用 SVD；
- 不显式计算 `(AᵀA)⁻¹`。

## 16. Gram-Schmidt 与 `A=QR`

Gram-Schmidt 把一组线性无关列转成同一列空间中的标准正交基。

![Gram-Schmidt 与 QR](https://oss.euler.icu/teaser/docs/algorithms-and-mathematics/linear-algebra-18-06/07-gram-schmidt-qr.png?v=20260711-final)

### 16.1 两列情形

对独立向量 `a₁,a₂`：

```text
q₁=a₁/||a₁||
u₂=a₂-(q₁ᵀa₂)q₁
q₂=u₂/||u₂||。
```

关键动作是：从 `a₂` 中减掉它在旧方向 `q₁` 上的分量，剩余部分必与 `q₁` 正交。

### 16.2 多列算法

```text
uⱼ=aⱼ-Σ_{i<j}(qᵢᵀaⱼ)qᵢ
qⱼ=uⱼ/||uⱼ||。
```

若某个 `uⱼ=0`，说明 `aⱼ` 可由前面列组合，列向量相关。

### 16.3 `R` 记录什么

每个原列都可由 `q₁,...,qⱼ` 表示：

```text
aⱼ=Σ_{i≤j} rᵢⱼqᵢ。
```

因此：

```text
A=QR，QᵀQ=I，R 为上三角矩阵。
```

`Q` 给列空间的正交骨架，`R` 给原列在这组骨架下的坐标。

### 16.4 用 QR 解最小二乘

若 `A=QR` 且 `A` 满列秩：

```text
min ||b-QRx||
```

正规条件化为：

```text
Rx=Qᵀb。
```

只需解上三角系统，无需形成 `AᵀA`。

### 16.5 经典算法与可靠实现

经典 Gram-Schmidt 在列近似相关时可能因舍入误差丢失正交性。实践中常用：

- Modified Gram-Schmidt；
- Householder 反射；
- Givens 旋转。

Householder QR 通常是稠密最小二乘的稳定默认选择。

## 17. 行列式：有向体积缩放

### 17.1 几何定义

`det(A)` 是线性变换 `A` 对有向体积的缩放因子：

- `|det(A)|` 是体积放大倍数；
- 符号表示方向是否翻转；
- `det(A)=0` 表示体积被压扁到更低维，矩阵不可逆。

二维中，`|det[a b]|` 是以 `a,b` 为边的平行四边形面积。

### 17.2 三条核心性质

行列式可由以下性质推出：

1. 对每一行线性；
2. 交换两行，符号反转；
3. `det(I)=1`。

由此得到消元规则：

| 行操作 | 行列式变化 |
| --- | --- |
| 两行交换 | 乘 `-1` |
| 某行乘 `c` | 乘 `c` |
| 某行加另一行的倍数 | 不变 |

### 17.3 乘法法则

```text
det(AB)=det(A)det(B)。
```

几何上，先由 `B` 缩放一次体积，再由 `A` 缩放一次，倍数相乘。

### 17.4 用 LU 计算

若 `PA=LU`：

```text
det(P)det(A)=det(L)det(U)。
```

单位下三角 `L` 的行列式为 1，`U` 的行列式是对角线乘积。因此实际计算大行列式应优先消元，而不是展开全部代数余子式。

### 17.5 行列式不告诉你什么

行列式只对方阵定义；它把许多结构压成一个数。`det(A)` 很小可能表示接近奇异，但不能精确告诉你哪个方向最脆弱；奇异值能提供更完整的信息。

## 18. 特征值与特征向量

### 18.1 定义背后的几何问题

我们寻找非零向量 `v`，使：

```text
Av=λv。
```

一般向量经过 `A` 后会改变方向；特征向量留在自己张成的直线上，只被缩放或反向。

![特征向量的几何意义](https://oss.euler.icu/teaser/docs/algorithms-and-mathematics/linear-algebra-18-06/08-eigenvectors.png?v=20260711-final)

### 18.2 如何计算

移项：

```text
(A-λI)v=0。
```

要有非零解，`A-λI` 必须奇异：

```text
det(A-λI)=0。
```

求根得到特征值；对每个 `λ` 求：

```text
N(A-λI)
```

得到特征空间。

### 18.3 例子

```text
A=[3 1]
  [1 3]。
```

特征多项式：

```text
det(A-λI)=(3-λ)²-1=(λ-4)(λ-2)。
```

所以：

```text
λ₁=4，v₁=(1,1)ᵀ
λ₂=2，v₂=(1,-1)ᵀ。
```

### 18.4 迹与行列式的快速检查

计代数重数时：

```text
Σᵢ λᵢ = trace(A)
Πᵢ λᵢ = det(A)。
```

上例中 `4+2=6=trace(A)`，`4·2=8=det(A)`。它们适合检查答案，但通常不是完整求解方法。

### 18.5 代数重数与几何重数

特征值在特征多项式中重复的次数是代数重数；特征空间维数是几何重数。总有：

```text
1 ≤ 几何重数 ≤ 代数重数。
```

若独立特征向量总数不足 `n`，矩阵不能对角化。

## 19. 对角化、矩阵幂与动力系统

### 19.1 为什么对角矩阵容易

若：

```text
A=XΛX⁻¹，
```

其中 `X` 的列是独立特征向量，`Λ` 是特征值对角矩阵，则：

```text
Aᵏ=XΛᵏX⁻¹。
```

中间因子全部抵消，矩阵幂转化为标量幂 `λᵢᵏ`。

### 19.2 坐标解释

对任意初始向量：

```text
x₀=c₁v₁+···+cₙvₙ，
```

反复应用 `A`：

```text
Aᵏx₀=c₁λ₁ᵏv₁+···+cₙλₙᵏvₙ。
```

长期行为由绝对值最大的特征值控制；对应特征向量给出主导方向。

### 19.3 差分方程

系统：

```text
x_{k+1}=Ax_k
```

的解为 `x_k=Aᵏx₀`。稳定性取决于谱半径：

```text
ρ(A)=max_i |λᵢ|。
```

- `ρ(A)<1`：所有模态衰减；
- `ρ(A)>1`：至少一个模态增长；
- `ρ(A)=1`：还需检查 Jordan 块和初始分量。

### 19.4 微分方程与矩阵指数

```text
u'(t)=Au(t)
```

的解为：

```text
u(t)=e^{At}u(0)。
```

若 `A=XΛX⁻¹`：

```text
e^{At}=Xe^{Λt}X⁻¹，
e^{Λt}=diag(e^{λ₁t},...,e^{λₙt})。
```

特征值的实部决定连续系统是衰减还是增长。

### 19.5 不能盲目对角化

并非每个方阵都有 `n` 个独立特征向量。遇到不可对角化矩阵时，可以使用 Jordan 形理解理论结构，但数值计算通常更偏好 Schur 分解，因为 Jordan 结构对扰动极其敏感。

## 20. 对称矩阵：线性代数中的好矩阵

### 20.1 谱定理

实对称矩阵 `A=Aᵀ` 有三个关键性质：

1. 所有特征值都为实数；
2. 不同特征值的特征向量正交；
3. 存在正交矩阵 `Q` 使

```text
A=QΛQᵀ。
```

与普通对角化相比，逆矩阵 `Q⁻¹` 简化为 `Qᵀ`。

### 20.2 为什么不同特征值对应正交向量

若 `Av=λv`、`Aw=μw`，利用对称性：

```text
vᵀAw=(Av)ᵀw。
```

所以：

```text
μvᵀw=λvᵀw
(μ-λ)vᵀw=0。
```

当 `λ≠μ`，必有 `vᵀw=0`。

### 20.3 二次型的主轴

令 `x=Qy`：

```text
xᵀAx=yᵀΛy=Σᵢ λᵢyᵢ²。
```

在特征向量坐标中，交叉项消失。对称矩阵的特征向量就是二次曲面的主轴。

### 20.4 为什么 `AᵀA` 总是重要

对任意实矩阵 `A`：

```text
AᵀA
```

总是对称半正定，因为：

```text
xᵀAᵀAx=||Ax||²≥0。
```

它的特征向量将成为右奇异向量，特征值是奇异值平方。

## 21. 正定矩阵与二次型

实对称矩阵 `A` 正定，是指：

```text
xᵀAx>0，对所有 x≠0。
```

![正定矩阵的代数、几何与优化解释](https://oss.euler.icu/teaser/docs/algorithms-and-mathematics/linear-algebra-18-06/09-positive-definite.png?v=20260711-final)

### 21.1 四种常用判定

对实对称矩阵，以下条件等价：

```text
xᵀAx>0，x≠0
⇔ 所有特征值 λᵢ>0
⇔ 所有顺序主子式为正
⇔ 存在可逆 R，使 A=RᵀR。
```

还可加入一个消元判据：不需要行交换时，`LDLᵀ` 分解中的所有主元都为正。

### 21.2 例子

```text
A=[2 1]
  [1 2]。
```

特征值为 `3,1`，均为正；顺序主子式为：

```text
Δ₁=2>0，Δ₂=det(A)=3>0。
```

Cholesky 分解可写为：

```text
R=[√2  1/√2]
  [ 0  √(3/2)]，

A=RᵀR。
```

### 21.3 几何：等值线为什么是椭圆

在特征坐标中：

```text
xᵀAx=λ₁y₁²+···+λₙyₙ²。
```

固定等值 `c` 时，第 `i` 个半轴长度为：

```text
sqrt(c/λᵢ)。
```

所以大特征值方向曲率大、半轴短；小特征值方向曲率小、半轴长。

### 21.4 优化：为什么有唯一极小值

二次函数：

```text
f(x)=1/2 xᵀAx-bᵀx
```

满足：

```text
∇f=Ax-b，∇²f=A。
```

若 `A` 正定，`f` 严格凸，方程 `Ax=b` 的解就是唯一全局极小点。

### 21.5 半正定与零方向

若只要求 `xᵀAx≥0`，则 `A` 半正定，允许零特征值。几何上碗底可能沿某些方向变成平坦谷底，极小点不再唯一。

## 22. 复向量、Fourier 与 FFT 思想

### 22.1 共轭转置

复向量空间中，普通转置要替换为共轭转置：

```text
zᴴw=Σᵢ conjugate(zᵢ)wᵢ。
```

长度为：

```text
||z||²=zᴴz。
```

满足 `QᴴQ=I` 的复矩阵称为酉矩阵；满足 `Aᴴ=A` 的矩阵称为 Hermitian 矩阵。它们分别对应实数情形的正交矩阵和对称矩阵。

### 22.2 Fourier 基

`n` 次单位根：

```text
ω=e^{-2πi/n}
```

生成离散 Fourier 矩阵。归一化后，Fourier 矩阵是酉矩阵，其列构成一组复标准正交基。

把信号 `x` 乘 Fourier 矩阵，相当于从“时间坐标”换到“频率坐标”：

```text
x = 时间样本
Fx = 各频率模式的系数。
```

### 22.3 为什么卷积在频率域变简单

循环卷积矩阵是 circulant 矩阵。Fourier 基是所有 circulant 矩阵共同的特征向量基，所以：

```text
C=FᴴΛF。
```

卷积 `Cx` 因而变成：

```text
先 Fourier 变换
逐频率相乘
再逆 Fourier 变换。
```

### 22.4 FFT 不是新的变换

DFT 是数学变换；FFT 是利用单位根对称性快速计算 DFT 的算法。直接矩阵乘法约需 `O(n²)`，FFT 将复杂度降为 `O(n log n)`。

这里再次出现课程主线：**选择让算子接近对角形的基，计算就会变简单。**

## 23. 相似矩阵与 Jordan 形

### 23.1 相似表示同一个线性变换

若：

```text
B=S⁻¹AS，
```

则 `A` 与 `B` 相似。它们是在不同基下表示同一个线性变换，因此共享：

- 特征值及其代数重数；
- 行列式和迹；
- 特征多项式；
- 秩。

但矩阵元素、特征向量坐标、奇异值通常不因相似而保持。

### 23.2 对角化是最理想的相似变换

若能找到由特征向量组成的基 `X`：

```text
X⁻¹AX=Λ。
```

这说明每个新坐标轴都是一个不变方向，变换在该坐标中完全解耦。

### 23.3 缺少特征向量时怎么办

例如：

```text
A=[1 1]
  [0 1]
```

只有特征值 `1`，但特征空间维数为 1，不能对角化。Jordan 形保留最接近对角的结构：

```text
J=[1 1]
  [0 1]。
```

上超对角线的 1 记录广义特征向量链。

### 23.4 Jordan 块如何影响矩阵幂

写 `J=λI+N`，其中 `N` 幂零。对 2 阶块：

```text
Jᵏ=λᵏI+kλ^{k-1}N。
```

所以仅看 `|λ|` 还不总够：当 `|λ|=1` 时，Jordan 块可能带来多项式增长。

### 23.5 理论工具与数值工具要区分

Jordan 形精确描述代数结构，但对微小扰动极敏感。数值线性代数通常计算 Schur 分解：

```text
A=QTQᵀ
```

实情形中 `T` 为准上三角矩阵。正交 `Q` 保持数值稳定，`T` 的对角块暴露特征值。

## 24. 奇异值分解：任意矩阵的标准坐标

特征分解主要服务于方阵，而且可能缺少足够特征向量。SVD 对任意 `m×n` 实矩阵都存在：

```text
A=UΣVᵀ。
```

![SVD 的旋转、拉伸、再旋转](https://oss.euler.icu/teaser/docs/algorithms-and-mathematics/linear-algebra-18-06/10-svd-geometry.png?v=20260711-final)

### 24.1 三个因子的几何意义

```text
Vᵀ：在输入空间中旋转或反射到右奇异向量坐标
Σ ：沿坐标轴按 σᵢ 拉伸，并可能压掉某些方向
U ：在输出空间中旋转或反射到最终方向。
```

所以 SVD 把任何线性变换分解成正交变换、轴向缩放、正交变换。

### 24.2 如何从 `AᵀA` 得到 SVD

`AᵀA` 对称半正定，因此有标准正交特征向量 `vᵢ`：

```text
AᵀAvᵢ=σᵢ²vᵢ，σᵢ≥0。
```

对 `σᵢ>0`，定义：

```text
uᵢ=Avᵢ/σᵢ。
```

于是：

```text
Avᵢ=σᵢuᵢ
Aᵀuᵢ=σᵢvᵢ。
```

### 24.3 紧致 SVD

若 `rank(A)=r`，只保留正奇异值：

```text
A=U_r Σ_r V_rᵀ，
```

其中：

```text
U_r∈R^{m×r}
Σ_r∈R^{r×r}
V_r∈R^{n×r}。
```

这比完整 SVD 更直接地暴露有效维数 `r`。

### 24.4 SVD 与四个基本子空间

```text
C(A)=span{u₁,...,u_r}
N(Aᵀ)=span{u_{r+1},...,u_m}
C(Aᵀ)=span{v₁,...,v_r}
N(A)=span{v_{r+1},...,v_n}。
```

SVD 不是课程末尾的孤立技巧。它为四个基本子空间同时提供标准正交基，并给每个有效方向配上强度 `σᵢ`。

### 24.5 SVD 与特征分解的区别

| 特征分解 | SVD |
| --- | --- |
| 主要针对方阵 | 任意矩形矩阵 |
| 左右使用同一个特征方向系统 | 输入方向 `vᵢ` 与输出方向 `uᵢ` 可不同 |
| 特征值可为负或复数 | 奇异值总为非负实数 |
| 可能不可对角化 | 总存在正交奇异向量基 |

对实对称正定矩阵，二者高度一致：奇异向量就是特征向量，奇异值就是正特征值。

## 25. 低秩近似、伪逆与条件数

### 25.1 秩一分层

紧致 SVD 可展开为：

```text
A=σ₁u₁v₁ᵀ+···+σ_ru_rv_rᵀ。
```

每一项都是秩一矩阵。SVD 按奇异值从大到小，把矩阵拆成越来越次要的结构层。

![SVD 低秩近似](https://oss.euler.icu/teaser/docs/algorithms-and-mathematics/linear-algebra-18-06/11-low-rank-approximation.png?v=20260711-final)

### 25.2 最佳秩 `k` 近似

截断到前 `k` 项：

```text
A_k=Σ_{i=1}^k σᵢuᵢvᵢᵀ。
```

Eckart-Young 定理说明，在所有秩不超过 `k` 的矩阵中，`A_k` 是谱范数和 Frobenius 范数下的最佳近似：

```text
||A-A_k||₂=σ_{k+1}
||A-A_k||_F²=Σ_{i>k}σᵢ²。
```

这给图像压缩、降维、去噪一个清晰解释：保留大奇异值对应的主要结构，丢弃小奇异值对应的弱细节或噪声。

### 25.3 Moore-Penrose 伪逆

若：

```text
A=UΣVᵀ，
```

把每个非零奇异值取倒数，得到：

```text
A⁺=VΣ⁺Uᵀ。
```

它统一了不同形状和秩的线性系统：

- 方阵可逆时，`A⁺=A⁻¹`；
- 超定系统中，`x=A⁺b` 给最小二乘解；
- 欠定且有多解时，`x=A⁺b` 给欧氏范数最小的解；
- 秩亏时，它自动忽略零奇异值方向。

### 25.4 满秩特殊公式

若 `A` 满列秩：

```text
A⁺=(AᵀA)⁻¹Aᵀ。
```

若 `A` 满行秩：

```text
A⁺=Aᵀ(AAᵀ)⁻¹。
```

这些是特殊情形，不应在秩亏时强行使用。

### 25.5 条件数

对满秩方阵：

```text
κ₂(A)=σ_max/σ_min。
```

它衡量最强拉伸与最弱拉伸的比例。若 `σ_min` 很小，某个输入方向几乎被压扁；逆问题需要除以这个小数，噪声就被大幅放大。

几何上：

```text
圆被映成越细长的椭圆，问题条件越差。
```

条件数描述问题本身对扰动的敏感度；算法稳定性描述计算过程是否额外放大误差。两者不要混为一谈。

## 26. 图、网络、Markov 链与差分方程

### 26.1 关联矩阵把图变成线性代数

给无向图任意指定边方向。令关联矩阵 `B` 的每行对应一条边，每列对应一个节点；一条从节点 `i` 指向 `j` 的边在第 `i` 列记 `-1`，第 `j` 列记 `+1`。

对节点势 `x`：

```text
Bx
```

给出每条边两端的势差。

### 26.2 零空间与连通分量

若图连通：

```text
B1=0，N(B)=span{1}。
```

所有节点势同时加同一个常数，不改变边差。若图有 `c` 个连通分量，`dim N(B)=c`，每个分量都可独立加常数。

这是一条极有代表性的翻译：**图的连通分量数量变成矩阵零空间维数。**

### 26.3 图 Laplacian

```text
L=BᵀB。
```

它对称半正定，因为：

```text
xᵀLx=||Bx||²=Σ_{边(i,j)}(x_i-x_j)²≥0。
```

连通图中：

```text
N(L)=span{1}。
```

第二小特征值反映图的连通强度，对应特征向量用于谱聚类。

### 26.4 流量与守恒

若 `f` 是边流量，`Bᵀf` 给每个节点的净流入。方程：

```text
Bᵀf=s
```

表示节点源汇守恒。总源汇必须为零，因为 `1ᵀBᵀf=(B1)ᵀf=0`。

### 26.5 Markov 矩阵

若采用列向量状态，并让转移矩阵 `P` 的每列和为 1，则：

```text
x_{k+1}=Px_k。
```

概率总和保持不变。稳态满足：

```text
Pπ=π，
```

所以 `π` 是特征值 `1` 的特征向量。其他特征值的绝对值决定收敛速度；若第二大模特征值接近 1，混合很慢。

### 26.6 应用题的共同模式

图、弹簧、电路、扩散、Markov 链看似不同，但常具有同一结构：

```text
局部差值：B
能量或扩散：BᵀB
守恒：左零空间
稳定状态：零特征值或特征值 1。
```

线性代数的价值不只是解某一个模型，而是识别跨领域重复出现的矩阵结构。

## 27. 五个矩阵分解的统一地图

Strang 在后续教材中常用五个分解组织线性代数。每个分解都回答一个不同问题。

### 27.1 `A=CR`：秩分解

从 `A` 选出 `r` 个独立列组成 `C`，其余每一列都由这些基列组合，组合系数组成 `R`：

```text
A=CR，
C∈R^{m×r}，R∈R^{r×n}。
```

它直接说明：

```text
rank(A)=r
C(A)=C(C)
N(A)=N(R)。
```

### 27.2 `PA=LU`：消元结构

```text
P：行交换
L：消元乘子
U：上三角结果。
```

用于高效解方阵系统、计算行列式和复用多个右端项。

### 27.3 `A=QR`：正交化结构

```text
Q：列空间的标准正交基
R：原列在该基下的坐标。
```

用于最小二乘、稳定求解和特征值算法。

### 27.4 `S=QΛQᵀ`：对称谱结构

对实对称矩阵：

```text
Q：正交特征向量
Λ：实特征值。
```

用于二次型、能量、优化、振动和主轴分析。

### 27.5 `A=UΣVᵀ`：任意矩阵的奇异结构

```text
V：输入正交方向
Σ：各方向强度
U：输出正交方向。
```

用于秩、四个子空间、伪逆、压缩、降维和病态分析。

### 27.6 如何记住而不背诵

| 你想知道什么 | 首选分解 |
| --- | --- |
| 哪些列真正独立 | `CR` |
| 如何重复解方阵系统 | `LU` |
| 如何得到正交列、做最小二乘 | `QR` |
| 对称变换有哪些主轴 | `QΛQᵀ` |
| 任意矩阵最强和最弱方向是什么 | `UΣVᵀ` |

这些不是五套互不相关的技巧，而是把同一个矩阵换成适合当前问题的坐标语言。

## 28. 陌生题的决策框架

### 28.1 第一步：识别对象形状

```text
A 是方阵、瘦高矩阵，还是矮宽矩阵？
A 是否对称？是否正定？是否稀疏？
问题要求精确解、最近解、最小范数解，还是长期行为？
```

形状不会单独决定答案，但会提示可能的结构。

### 28.2 第二步：翻译问题

| 原问题语言 | 线性代数翻译 |
| --- | --- |
| 能否达到目标 `b` | `b∈C(A)`？ |
| 哪些输入没有影响 | 求 `N(A)` |
| 有多少自由参数 | `n-r` |
| 数据拟合 | 投影到 `C(A)` |
| 最近点 | 正交残差 |
| 反复迭代 | 特征值、对角化、Jordan/Schur |
| 二次能量是否总为正 | 正定性 |
| 压缩或降维 | 截断 SVD |
| 解对噪声是否敏感 | 最小奇异值与条件数 |

### 28.3 第三步：选择结构而不是公式

```text
方阵求解且结构一般       → 带选主元 LU
对称正定系统             → Cholesky
超定最小二乘             → QR
秩亏或病态最小二乘       → SVD
矩阵幂、线性动力系统      → 特征/Schur 分解
只需子空间和秩           → 消元或 rank-revealing QR/SVD。
```

### 28.4 第四步：做维数检查

每一步都写出尺寸。例如 `A∈R^{m×n}`：

```text
x∈Rⁿ，b∈Rᵐ
AᵀA∈R^{n×n}
AAᵀ∈R^{m×m}
N(A)⊆Rⁿ
N(Aᵀ)⊆Rᵐ。
```

大量错误无需计算，只靠维数就能发现。

### 28.5 第五步：用不变量检查

- `rank(A)` 是否与主元个数一致？
- `dim N(A)=n-r` 是否成立？
- 特征值和是否等于迹、积是否等于行列式？
- 投影矩阵是否满足 `P²=P,Pᵀ=P`？
- 正交矩阵是否满足 `QᵀQ=I`？
- SVD 是否满足 `Avᵢ=σᵢuᵢ`？
- 最小二乘残差是否满足 `Aᵀe=0`？

### 28.6 证明题的常用入口

| 要证明的结论 | 常用入口 |
| --- | --- |
| 子空间 | 检查对线性组合封闭 |
| 线性无关 | 令线性组合为零 |
| 正交 | 计算点积 |
| 投影 | 验证像空间、幂等、正交残差 |
| 正定 | 写成平方和或检查特征值 |
| 秩关系 | 比较零空间、列空间或使用分解 |
| 特征值性质 | 从 `Av=λv` 出发 |

真正成熟的解题习惯是：先找定义最短路径，再决定是否需要计算。

## 29. 常见误区与反例

### 29.1 “矩阵乘法就是逐元素相乘”

错误。普通矩阵乘法来自变换复合和行列点积；逐元素乘积是 Hadamard 乘积，是另一种操作。

### 29.2 “`AB=AC` 可以约掉 `A`”

只有 `A` 有左逆时才能推出 `B=C`。若 `A` 有非零零空间，`A(B-C)=0` 不代表 `B-C=0`。

### 29.3 “有 `n` 个特征值就一定可对角化”

按代数重数总有 `n` 个复特征值，但可能没有 `n` 个独立特征向量。Jordan 块就是反例。

### 29.4 “主元列取自 RREF”

主元**位置**从 RREF 读，列空间的基要回原矩阵取对应列。行操作保持行空间和零空间，但一般不保持列空间中的具体列。

### 29.5 “`AᵀA` 总可逆”

只有 `A` 满列秩时才可逆。事实上：

```text
N(AᵀA)=N(A)。
```

### 29.6 “最小二乘让 `Ax=b` 成立”

它让 `Ax` 尽可能接近 `b`。真正成立的是：

```text
AᵀA x̂=Aᵀb，
```

而不是原方程。

### 29.7 “零行列式只是一种计算失败”

`det(A)=0` 表示一个几何方向被压扁，列相关、零空间非平凡、变换不可逆。这是结构事实，不是算法偶然失败。

### 29.8 “特征值描述所有矩形矩阵”

矩形矩阵没有通常意义下的特征值。应使用奇异值；它们描述输入到输出的拉伸强度。

### 29.9 “对称就一定正定”

对称只保证实特征值和正交特征向量。若有负特征值，二次型在某些方向为负；若有零特征值，只是半正定。

### 29.10 “正规方程总是最佳实现”

正规方程是正确理论，但会平方条件数。精度重要时优先 QR 或 SVD。

### 29.11 “奇异值小就是没有信息”

小奇异值可能是噪声，也可能是应用中关键但微弱的方向。是否截断取决于噪声尺度和任务目标，不能只按大小机械删除。

### 29.12 “坐标就是向量”

坐标依赖基。换基后数字改变，但抽象向量和线性变换不变。

## 31. 自测题与答案线索

### 31.1 基础结构

**题 1**：对 `A∈R^{4×7}` 且 `rank(A)=3`，写出四个基本子空间所在环境和维数。

答案线索：

```text
dim C(A)=3，dim N(Aᵀ)=1；
dim C(Aᵀ)=3，dim N(A)=4。
```

**题 2**：为什么 RREF 的非零行可以作行空间的基，而主元列必须回原矩阵取？

答案线索：行操作是行的可逆线性组合，保持行空间；却会改变具体列向量。

**题 3**：若 `N(A)={0}`，能否立即断言 `Ax=b` 对每个 `b` 有解？

答案线索：只有当 `A` 是方阵，或同时知道 `rank(A)=m` 时才行。对瘦高矩阵，满列秩只保证至多一解，不保证每个 `b∈Rᵐ` 可达。

### 31.2 正交与最小二乘

**题 4**：证明投影矩阵 `P=A(AᵀA)⁻¹Aᵀ` 满足 `P²=P` 与 `Pᵀ=P`。

答案线索：直接代入，并使用 `(AᵀA)ᵀ=AᵀA`。

**题 5**：若 `Q` 有标准正交列，证明投影到 `C(Q)` 的矩阵是 `QQᵀ`。

答案线索：在一般投影公式中代入 `QᵀQ=I`。

**题 6**：最小二乘残差为什么位于 `N(Aᵀ)`？

答案线索：最近点的一阶条件是 `Aᵀ(b-Ax̂)=0`。

### 31.3 特征值与正定性

**题 7**：若 `A²=A`，它的特征值可能是什么？

答案线索：对 `Av=λv` 应用 `A²v=Av`，得到 `λ²=λ`，所以 `λ∈{0,1}`。

**题 8**：若 `Q` 正交，它的实特征值可能是什么？复特征值模长是多少？

答案线索：`||Qv||=||v||`，因此 `|λ|=1`；实特征值只有 `±1`。

**题 9**：为什么 `AᵀA` 不可能有负特征值？

答案线索：若 `AᵀAv=λv`，则 `λ||v||²=||Av||²≥0`。

**题 10**：构造一个对称但不正定的矩阵。

答案线索：`diag(1,-1)` 对称，但二次型在第二坐标方向为负。

### 31.4 SVD 与应用

**题 11**：若奇异值为 `10,1,0.01`，谱范数条件数是多少？截断到秩 2 的谱范数误差是多少？

答案线索：`κ₂=1000`，误差 `σ₃=0.01`。

**题 12**：证明 `N(A)=N(AᵀA)`。

答案线索：一个方向显然；另一个方向由 `xᵀAᵀAx=||Ax||²=0` 得到。

**题 13**：为什么 `rank(A)` 等于正奇异值个数？

答案线索：SVD 中只有正奇异值对应的输入方向被映到非零、独立的输出方向。

**题 14**：连通图的 Laplacian 为什么至少有一个零特征值？

答案线索：`L1=BᵀB1=0`。

**题 15**：Markov 链为什么会出现特征值 1？

答案线索：概率守恒使 `1ᵀP=1ᵀ`；在适当条件下存在右特征向量 `π` 满足 `Pπ=π`。
